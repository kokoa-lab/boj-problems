## 문제

포켓몬 GO에서 포켓몬을 진화시키려면 사탕이 필요하다.

포켓몬의 번호가 주어졌을 때, 이 포켓몬을 진화시키려면 사탕이 몇 개 필요한지 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 포켓몬의 번호 N (1 ≤ N ≤ 151)이 주어진다.

## 출력

첫째 줄에 입력으로 주어진 N번 포켓몬이 진화하는데 필요한 사탕의 개수를 출력한다. 만약, 진화하지 않는 포켓몬인 경우에는 -1을 출력한다. 2017년 2월 이후에 추가된 포켓몬으로 진화하는 포켓몬은 진화하지 않는 포켓몬이다.
