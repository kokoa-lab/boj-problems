## 문제

앞 사람이 제출한 수 보다 1 큰 수를 제출하면 페널티를 100 감소시켜 준다.

앞 사람은 자기 자신을 제외한다.

출력한 수 만큼 점수를 받게 된다.

## 입력

입력은 없다.

## 출력

첫째 줄에 정수 N을 출력한다. (1 ≤ N ≤ 100,000)
