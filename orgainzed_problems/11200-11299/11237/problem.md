## 문제

aทุกๆ ปีจะมีการสอบแข่งขันเพื่อค้นหา “สัตว์ที่ฉลาดที่สุดในจักรวาล” การสอบคัดเลือกรอบแรกจะมีการจัดขึ้นที่อาณาจักรของสัตว์แต่ละชนิด เพื่อคัดเลือกตัวแทนที่มีความรู้ความสามารถที่หลากหลาย ไม่ว่าจะเป็น ความรู้ด้านคณิตศาสตร์ วิทยาศาสตร์ สังคม ปรัชญา ศิลปะ ดนตรี และศาสตร์แขนงอื่นๆ อีกมากมายโดยรูปแบบการสอบคัดเลือกรอบแรกนั้นจะแตกต่างกันไปตามที่แต่ละอาณาจักรเป็นผู้ก าหนดที่อาณาจักรกระต่ายก็มีการสอบคัดเลือกรอบแรกเพื่อค้นหากระต่ายอัจฉริยะที่แฝงตัวอยู่ทั่วอาณาจักร

เพื่อไปสอบแข่งขันในรอบต่อไป ส าหรับการสอบคัดเลือกรอบแรกของอาณาจักรกระต่ายนั้น กรรมการคัดเลือกจะออกข้อสอบเป็นแบบปรนัยชนิดเลือกตอบ (Multiple Choice Exam) เพื่อให้สะดวกในการตรวจ โดยกรรมการคัดเลือกได้สรรหาผู้ทรงคุณวุฒิท าการออกข้อสอบท่านละ 1 ชุด ผู้เข้าสอบต้องเลือกท าเพียงชุดใดชุดหนึ่ง ผู้ที่ได้คะแนนสูงสุด 50 อันดับแรกจะเป็นตัวแทนไปแข่งขันในรอบต่อไป

หากคุณเป็นกระต่ายตัวหนึ่งที่ต้องเข้าร่วมการสอบคัดเลือกรอบแรกนี้ คุณจะเลือกท าข้อสอบชุดใดเพื่อให้มีความน่าจะเป็นที่จะได้คะแนนสอบสูงสุด

## 입력

บรรทัดแรกเป็นจ านวนกรณีทดสอบ T ชุด (1 ≤ T ≤ 20) กรณีทดสอบแต่ชุดประกอบด้วยข้อมูลดังนี้

1. บรรทัดที่ 1 ของแต่ละกรณีเป็นจ านวนเต็ม N แทนจ านวนชุดข้อสอบ (1 < N ≤ 100)
2. ถัดไป N บรรทัดเป็นข้อมูลของชุดข้อสอบ N ชุด ตั้งแต่ชุดที่ 1 ถึง N โดยประกอบไปด้วย
   * จ านวนเต็ม M แทนจ านวนรูปแบบข้อสอบ (1 ≤ M ≤ 100)
   * ถัดไปอีก M บรรทัด แสดงข้อมูลของรูปแบบข้อสอบที่ปรากฏอยู่ในชุดข้อสอบนั้นประกอบด้วย จ านวนเต็ม X, Y, Z โดยที่ X แทนจ านวนตัวเลือกในแต่ละข้อ, Y แทนคะแนนเต็มของแต่ละข้อ และ Z แทนจ านวนข้อ (1 < X ≤ 1 000 และ 1 ≤ Y, Z ≤ 100)

## 출력

ประกอบไปด้วย T บรรทัด แต่ละบรรทัดประกอบด้วยจ านวนเต็ม J แสดงหมายเลขชุดข้อสอบที่มีความน่าจะเป็นที่จะได้คะแนนสอบสูงสุด ถ้ามีชุดข้อสอบที่มีความน่าจะเป็นที่จะได้คะแนนสูงสุดเท่ากันมากกว่า 1 ชุด ให้แสดงหมายเลขชุดข้อสอบที่มีจ านวนข้อน้อยที่สุด และถ้าชุดข้อสอบดังกล่าวมีจ านวนข้อเท่ากัน ให้แสดงหมายเลขชุดข้อสอบที่น้อยที่สุด

## 힌트
