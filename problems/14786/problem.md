## 문제

A, B, C가 주어졌을 때, Ax+Bsin(x)=C를 만족하는 x를 찾는 프로그램을 작성하시오.

## 입력

첫째 줄에 정수 A, B, C가 주어진다. (0 < B ≤ A ≤ 100,000, 0 < C ≤ 100,000)

## 출력

첫째 줄에 x를 출력한다. 정답과의 절대/상대 오차는 10-9까지 허용한다.
