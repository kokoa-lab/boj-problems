## 문제

Чтобы космический корабль мог взлететь, каждому члену экипажа нужно выдать билетик. На каждом билетике должна быть написана последовательность из $n$ цифр, где $n$ --- четное положительное число.

Даниил считает билет супер-счастливым, если выполняются два свойства:

* Сумма первых $\frac{n}{2}$ цифр равна сумме последних $\frac{n}{2}$ цифр.
* Сумма цифр на четных позициях равна сумме цифр на нечетных позициях.

Помогите Даниилу посчитать количество различных супер-счастливых билетиков по модулю $998\,244\,353$.

## 입력

В первой строке дано одно число $n$ --- длина билетика ($2 \le n \le 200\,000$). Гарантируется, что $n$ чётное.

## 출력

Выведите количество супер-счастливых билетиков длины $n$ по модулю $998\,244\,353$.
