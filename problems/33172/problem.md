## 문제

長さ N の文字列 S が与えられる．S の各文字は英小文字である．

ある文字列 T および整数 m (2 ≦ m) が存在して，空文字列に対して T を m 回順に繋げた文字列が S に一致する時，S は**周期的**であるとする．

S が周期的であるなら `Yes` を，そうでないならば `No` を出力せよ．

## 입력

入力は以下の形式で与えられる．

```

N
S
```

## 출력

S が周期的であるなら `Yes` を，そうでないならば `No` を出力せよ．

答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
