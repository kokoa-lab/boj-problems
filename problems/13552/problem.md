## 문제

3차원 좌표 위에 점 N개가 있다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.

* `x, y, z, r`: 중심이 (x, y, z)이고, 반지름이 r인 구의 내부에 점이 몇 개 있는지 출력한다. 구의 표면 위에 있는 점도 내부에 있는 점이다.

## 입력

첫째 줄에 점의 개수 N (1 ≤ N ≤ 100,000)이 주어진다.

둘째 줄부터 N개의 줄에 점의 좌표 x, y, z가 주어진다. (0 ≤ x, y, z ≤ 1,000,000)

다음 줄에 쿼리의 개수 M이 주어진다. (1 ≤ M ≤ 100,000)

다음 M개의 줄에 쿼리의 정보 x, y, z, r이 주어진다. (0 ≤ x, y, z ≤ 1,000,000, 1 ≤ r ≤ 1,000,000)

## 출력

각각의 쿼리마다 정답을 한 줄에 하나씩 출력한다.
