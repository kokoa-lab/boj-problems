## 문제

**메모리 제한에 유의하십시오.**

길이가 $N$인 수열 $a$가 있다. 초기 상태에서 모든 $1 \le i \le N $에 대해 $a\_i = 0$을 만족한다. 이 $a$에 다음과 같은 아래 쿼리를 처리하는 프로그램을 작성하라.

* `1 L R w` : $L\le i\le R$ 인 $a\_i$에 $w$를 더한다.
* `2 L R` : $L \le i \le R$인 $a\_i$중 홀수와 짝수의 개수를 차례대로 하나의 줄에 공백으로 구분하여 출력한다.

## 입력

첫 번째 줄에 수열의 길이 $N$, 쿼리의 개수 $Q$가 공백으로 구분되어 주어진다.

두 번째 줄부터 $Q$개의 줄에 쿼리가 한 줄에 하나씩 주어진다. 2번 쿼리는 반드시 하나 이상 주어진다.

## 출력

각각의 2번 쿼리마다 정답을 한 줄에 하나씩 출력한다.
