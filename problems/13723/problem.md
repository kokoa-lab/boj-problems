## 문제

양의 정수 N이 주어졌을 때, 아래 식을 만족하는 양의 정수 해의 개수를 구하는 프로그램을 작성하시오.

\[\frac{1}{N!} = \frac{1}{X} + \frac{1}{Y}\]

## 입력

첫째 줄에 N (N ≤ 104)이 주어진다.

## 출력

해의 개수를 출력한다.
