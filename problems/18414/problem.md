## 문제

整数 X, L, R が与えられる. L 以上 R 以下の整数のうち，X との差の絶対値が最も小さいものを出力せよ．そのような整数はちょうど 1 つだけ存在することが証明できる．

## 입력

入力は以下の形式で標準入力から与えられる．

```

X L R
```

## 출력

L 以上 R 以下の整数のうち，X との差の絶対値が最も小さいものを出力せよ．
