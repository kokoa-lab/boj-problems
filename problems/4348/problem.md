## 문제

길이가 다양한 N개의 막대가 있다. 이때, 막대를 모두 사용해서 정사각형을 만들 수 있을까?

## 입력

첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있다. 각 줄의 첫 번째 숫자는 막대의 개수 N이다. (4 ≤ N ≤ 20) 다음 N개의 숫자는 막대의 길이이다. 막대의 길이는 1과 10,000사이의 자연수이다.

## 출력

각 테스트 케이스 마다 막대를 모두 사용해서 정사각형을 만들 수 있으면 yes를, 아니면 no를 출력한다.
