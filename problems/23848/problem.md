## 문제

등비수열이란 다음과 같이 연속한 두 항의 비가 일정한 수열이다.

* 1, 2, 4, 8, 16, 32...
* 4, 12, 36, 108, 324...

모든 항이 양의 정수인 등비수열 중, 수열의 합이 *N*이고 항이 3개 이상이며 공비가 1보다 큰 양의 정수인 등비수열을 구하여라.

가능한 답이 여러 개가 존재하면 그 중 하나를 출력하고, 가능한 답이 없다면 -1을 출력한다.

## 입력

등비수열의 합을 나타내는 정수 *N*이 주어진다. (1 ≤ *N* ≤ 1012)

## 출력

문제의 조건을 만족하는 등비수열이 존재하는 경우, 첫째 줄에 항의 개수 *K*(*K* ≥ 3)를 출력한다. 둘째 줄에 등비수열을 순서대로 공백으로 구분하여 출력한다.

조건을 만족하는 등비수열이 존재하지 않는 경우 -1을 출력한다.
