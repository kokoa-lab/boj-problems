## 문제

작도왕 키위새는 삼각형을 좋아한다. 2차원 평면 위의 점 $N$개가 주어질 때, 모든 점을 지나는 유일한 삼각형을 찾아보자.

## 입력

첫째 줄에 점의 수 $N$이 주어진다. ($1 \le N \le 500\,000$)

둘째 줄부터 $N$개의 줄에 각 점의 x, y좌표를 나타내는 정수 $x$와 $y$가 주어진다. ($-1\,000\,000 \le x, y \le 1\,000\,000$)

모든 점의 좌표는 서로 다르다.

## 출력

첫째 줄에 모든 점을 지나는 삼각형의 개수를 출력한다. 무한히 많을 경우 개수 대신 `-1`을 출력한다.

모든 점을 지나는 삼각형이 유일하다면, 둘째 줄부터 넷째 줄까지 삼각형의 각 변의 기울기를 출력한다.

기울기의 절대 / 상대 오차는 $10^{-6}$까지 허용한다. 삼각형이 유일하면서 어느 한 변의 기울기가 양의 무한대나 음의 무한대인 경우는 주어지지 않는다.
