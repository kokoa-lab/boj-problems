## 문제

JOI 君は，一直線の道路上で散歩をしている．

JOI 君は以下の行動を行動 A → 行動 B → 行動 A → … のように交互に繰り返す．

* 行動 A：3 m 前に進む．
* 行動 B：2 m 後ろに戻る．

行動 A と行動 B を合わせて X 回行ったとき，はじめの位置から何 m 前に移動したことになるか？

## 입력

入力は以下の形式で与えられる．

```

X
```

## 출력

はじめの位置から前に移動した距離が何 m かを，単位 (m) を除いて出力せよ．

答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
