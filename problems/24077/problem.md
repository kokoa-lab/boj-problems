## 문제

長さ N の整数列 A = (A1, A2, ..., AN) と長さ M の整数列 B = (B1, B2, ..., BM) が与えられる．

次の条件をすべて満たす 2 つの整数の組 (i,j) の個数を求めよ．

* 1 ≦ i ≦ N．
* 1 ≦ j ≦ M．
* Ai ≦ Bj．

## 입력

入力は以下の形式で標準入力から与えられる．

```

N M
A1 A2 ... AN
B1 B2 ... BM
```

## 출력

Ai ≦ Bj を満たす (i,j) の個数を出力せよ．
