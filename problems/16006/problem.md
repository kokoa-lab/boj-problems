## 문제

Consider a natural number N and the expression HN = 1 + 1/2 + 1/3 + ... + 1/N

Determine the natural numbers, the numerator P and the denominator Q of the irreducible fraction HN = P/Q.

## 입력

The standard input contains on the first line the number N.

## 출력

The standard output will contain on the first line the number P and on the second line the number Q.
