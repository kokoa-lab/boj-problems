## 문제

당신은 숫자 $k$를 싫어한다. 그래서 수를 쓸 때, 숫자 $k$는 쓰지 않으려 한다.

$f(x)$를, $x$의 10진법 표현에서 숫자 $k$를 모두 지운 수로 정의하자. 만일 $x$의 10진법 표현이 숫자 $k$로만 이루어져 있다면 $f(x)$를 $0$으로 정의한다.

예를 들어, $k = 3$일 때 $f(3135) = 15$, $f(2345) = 245$, $f(8185) = 8185$, $f(333333) = 0$이다.

숫자 $k$와 수 $N$이 주어질 때, $f(1) + f(2) + ... + f(N)$을 $1\,000\,000\,007$로 나눈 나머지를 구하여라.

## 입력

첫째 줄에 숫자 $k$가 주어진다.

둘째 줄에 수 $N$이 주어진다.

## 출력

첫째 줄에 $f(1) + f(2) + ... + f(N)$을 $1\,000\,000\,007$로 나눈 나머지를 출력하라.
