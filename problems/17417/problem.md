## 문제

다음은 교준이가 작성한 소스 코드다.

![](./001_preview)

**<그림 1>** 교준이가 작성한 소스 코드

다음은 교준이가 작성한 프로그램의 입력이다:

```

Q
N1 S1 E1
N2 S2 E2
...
NQ SQ EQ
```

교준이가 작성한 프로그램의 출력 결과를 출력하는 프로그램을 작성하시오.

## 입력

첫 번째 줄에 자연수 *Q*가 주어진다.

(i+1)번째 줄에는 세 자연수 *N*i, *S*i, *E*i가 사이에 공백을 두고 주어진다(1 ≤ i ≤ *Q*).

## 출력

첫 번째 줄부터 *Q*개의 줄에 걸쳐, 교준이의 프로그램의 출력 결과를 출력한다.
