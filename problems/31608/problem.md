## 문제

整数 N と，2 つの長さ N の 文字列 S, T が与えられる．S と T のハミング距離を出力せよ．

ここで S と T のハミング距離とは，S の i 文字目と T の i 文字目が異なるような i (1 ≦ i ≦ N) の個数である．

## 입력

入力は以下の形式で与えられる．

```

N
S
T
```

## 출력

S と T のハミング距離を出力せよ．

結果以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
