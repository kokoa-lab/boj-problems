## 문제

세 정수 $A\_1$, $A\_2$, $A\_3$이 주어질 때, 다음 조건을 만족하는 루트 있는 이진 트리의 개수를 구하라.

* 차수가 $1$인 정점의 개수가 $A\_1$개, 차수가 $2$인 정점의 개수가 $A\_2$개, 차수가 $3$인 정점의 개수가 $A\_3$개이다.

이때 루트 있는 이진 트리는 좌/우 자식을 구분하며, 정점에는 번호가 부여되지 **않는다**.

두 루트 있는 이진 트리가 서로 다르게 세어질 조건은 다음과 같은 정점이 하나 이상 있는 경우이다.

1. 한 트리의 루트에서 그 트리의 어떤 다른 정점까지 가기 위한 좌/우 순서열을 기록한다.
2. 다른 트리에서 해당 순서로 이동했을 때, 해당 위치에 정점이 없다.

예시로, 다음 두 트리를 보자.

|  |  |
| --- | --- |
| 왼쪽 트리 A | 오른쪽 트리 B |

두 트리 모두 $(A\_1,A\_2,A\_3) =(3,4,1)$인 트리이다. 두 트리의 루트는 $R$로 표기되어 있다. 왼쪽 트리는 왼쪽 $\rightarrow$ 오른쪽 $\rightarrow$ 왼쪽 경로를 따라갔을 때 정점 $x$가 있지만, 오른쪽 트리는 해당 경로를 따라갔을 때 도달하는 정점이 없기 때문에 서로 다른 트리로 세어진다.

## 입력

첫째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1\leq T\leq 100\, 000)$

다음 $T$개의 줄에 걸쳐 테스트 케이스가 주어진다. 각 테스트 케이스는 한 줄로 구성되며, 세 정수 $A\_1,A\_2,A\_3$가 공백으로 구분되어 주어진다. $(0\leq A\_1,A\_2,A\_3\leq 1\, 000\, 000;A\_1+A\_2+A\_3>0)$

## 출력

각 테스트 케이스마다 한 줄에 문제에서 요구하는 트리의 개수를 $998\, 244\, 353$으로 나눈 나머지를 출력한다.
