## 문제

Приготовление десертов можно считать ремеслом, можно считать искусством, а можно считать и наукой. В этой задаче мы будем использовать третий из указанных подходов.

Одна из кофеен предлагает своим посетителям большой выбор десертов, каждый из которых состоит из кусочка пирога с начинкой, который сверху украшен шариком мороженого. При этом тесто пирога может быть приготовлено одним из *n* способов, существует *m* вариантов начинки и *k* сортов мороженого. К сожалению, не все типы теста сочетаются со всем начинками, не все начинки — со всеми сортами мороженого, и не все сорта мороженого — со всеми типами теста.

Ваша задача состоит в определении числа типов десертов, которые могут быть приготовлены так, чтобы все три ингредиента сочетались друг с другом.

## 입력

Первая строка содержит три целых числа: *n*, *m* и *k* (1 ≤ *n*, *m*, *k* ≤ 50).

Вторая строка содержит одно целое число *p* (0 ≤ *p* ≤ 200) — количество пар типов теста и начинок, не сочетающихся друг с другом. Каждая из последующих *p* строк содержит по два целых числа *a* и *b* (1 ≤ *a* ≤ *n*, 1 ≤ *b* ≤ *m*) — номера типа теста и начинки, которые не сочетаются друг с другом. Никакая пара несочетающихся типов теста и начинки не указана более одного раза.

Следующая строка содержит одно целое число *q* (0 ≤ *q* ≤ 200) — количество пар начинок и сортов мороженого, не сочетающихся друг с другом. Каждая из последующих *q* строк содержит по два целых числа *a* и *b* (1 ≤ *a* ≤ *m*, 1 ≤ *b* ≤ *k*) — номера начинки и сорта мороженого, которые не сочетаются друг с другом. Никакая пара несочетающихся начинки и сорта мороженого не указана более одного раза.

Следующая строка содержит одно целое число *r* (0 ≤ *r* ≤ 200) — количество пар типов теста и сортов мороженого, не сочетающихся друг с другом. Каждая из последующих *r* строк содержит по два целых числа *a* и *b* (1 ≤ *a* ≤ *n*, 1 ≤ *b* ≤ *k*) — номера типа теста и сорта мороженого, которые не сочетаются друг с другом. Никакая пара несочетающихся типа теста и сорта мороженого не указана более одного раза.

## 출력

Выведите одно число — число типов десертов, которые могут быть приготовлены так, чтобы все три ингредиента сочетались друг с другом.
