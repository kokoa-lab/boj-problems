## 문제

양의 정수 $N$과 $K$가 주어질 때, $1$ 이상 $N$ 이하의 정수 중에서 서로 다른 $K$개의 정수를 적절히 선택하여, $K$개의 정수의 합이 $K+1$의 배수가 되도록 하는 것이 가능한지 판별해 보자.

## 입력

첫째 줄에 정수 $N$과 $K$가 공백으로 구분되어 주어진다. $(1 \le K \le N \le {10}^6 )$

## 출력

$1$ 이상 $N$ 이하의 정수 중에서 서로 다른 $K$개의 정수를 적절히 선택하여, $K$개의 정수의 합이 $K+1$의 배수가 되도록 하는 것이 가능하면 `YES`, 불가능하면 `NO`를 출력한다.
