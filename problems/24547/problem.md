## 문제

길이가 $N$인 양의 정수들의 수열 $A\_1, A\_2, ... A\_N$이 주어진다. 이때, 다음과 같은 쿼리를 수행하는 프로그램을 작성하시오.

* $1$ $X$ : 주어지는 양의 정수 $X$에 대해 $ \sum\_{i=1}^N {({A\_i}\bmod{X})} $를 출력한다.
* $2$ $X$ : 주어지는 양의 정수 $X$에 대해 $ \sum\_{i=1}^N {({X}\bmod{A\_i})} $를 출력한다.
* $3$ $i$ $X$ : $A\_i$의 값을 양의 정수 $X$로 바꾼다. ($1\le i \le N$)

${P}\bmod{Q}$는 $P$를 $Q$로 나눈 나머지를 뜻한다.

## 입력

첫째 줄에 수열의 길이 $N$이 주어진다. ($1\le N \le 100\,000$)

둘째 줄에는 양의 정수들의 수열 $A\_1, A\_2, ... A\_N$이 주어진다. ($1\le A\_i \le 100\,000$)

셋째 줄에는 쿼리의 개수 $Q$가 주어진다. ($1\le Q \le 100\,000$)

넷째 줄부터 $Q$개의 줄에 걸쳐 쿼리가 주어진다. ($1\le X \le 100\,000$)

## 출력

1, 2번 쿼리에 대해서 정답을 한 줄에 하나씩 출력한다.
