## 문제

![](./001_XOR.png) *S*에서 *F*까지의 모든 정수를 XOR한 값은 무엇일까?

## 입력

입력의 첫 번째 줄에는 테스트 케이스의 개수 *T*(1 ≤ *T* ≤ 1000)가 주어진다.

다음 *T* 개의 줄에는 두 개의 정수 *S*와 *F*가 주어진다. (1 ≤ *S* ≤ *F* ≤ 1 000 000 000)

## 출력

각 테스트 케이스마다 *S*에서 *F*까지의 모든 정수를 XOR한 값을 출력한다.
