## 문제

양의 정수 S와 X가 주어질 때, A + B = S, A ⊕ B = X을 만족하는 모든 양의 정수 A, B 쌍의 개수를 구하시오.

## 입력

첫 줄에는 두 양의 정수 S와 X가 주어진다.(2 ≤ S, X ≤ 1012)

## 출력

위 식을 만족하는 모든 A, B 쌍의 개수를 구하시오.
