## 문제

Пусть задано некоторое множество точек на плоскости. *Выпуклая оболочка* --- наименьший выпуклый многоугольник, содержащий данные точки.

![](./001_preview)

Необходимо найти число точек с целыми координатами, которые находятся строго внутри выпуклой оболочки заданного множества точек.

## 입력

Первая строка входного файла содержит целое число $n$ ($3 \le n \le 100\,000$). Каждая из последующих $n$ строк описывает одну точку и содержит по два целых числа $x\_i$ и $y\_i$ ($|x\_i|, |y\_i| \le 10^9$) --- ее координаты. Никакие две точки не совпадают. Среди заданных точек есть три, не лежащие на одной прямой.

## 출력

В выходной файл выведите ответ на задачу.
