## 문제

2차원 평면 상에 N개의 점이 주어진다. 각각의 점에 대해서, 해당 점을 제외하고 모든 점을 포함하는 최소 넓이의 볼록 다각형을 구하라. 어느 세 점도 한 직선 위에 존재하지 않는다.

## 입력

첫 번째 줄에는 정수 N이 주어진다. (4 ≤ N ≤ 100,000)

이후 N개의 줄에 점의 좌표 xi, yi가 주어진다. (0 ≤ xi, yi ≤ 1,000,000,000)

## 출력

i번째 줄에, 해당 점이 빠졌을 때 다각형의 넓이를 소수점 셋째 자리까지 출력한다.
