## 문제

長さ N の文字列 S が与えられる．S の各文字は `A`，`B`，`C`，`D`，`E` のいずれかである．

S に 3 種類以上の文字が出現する場合は `Yes` を，そうでない場合は `No` を出力せよ．

## 입력

入力は以下の形式で標準入力から与えられる．

```

N
S
```

## 출력

S に 3 種類以上の文字が出現する場合は `Yes` を，そうでない場合は `No` を出力せよ．
