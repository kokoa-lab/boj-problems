## 문제

가희는 업힐과 다운힐을 좋아하는 자전거 라이더다. 가희는 위치 *X*0, 높이 *H*0인시작지점에서 출발하여 위치 *Xi*와 높이 *Hi*가 서로 같거나 다른 *N*개의 체크 포인트를 주행할 계획을 하고 있다. 각각의 체크 포인트는 시작지점과 수평거리가 가까운 순서대로 1, 2, 3, ..., *N*이라고 불리우며, 시작지점을 포함한 인접한 체크 포인트들은 서로 일직선으로 연결되어 있다.

![](./001_preview)

가희는 안전한 라이딩을 위해서, 구간별 평균 기울기를 조사하려고 한다. 그래야 어느 지점에서 브레이크를 어떤 강도로 잡을지 계획을 세울 수 있기 때문이다. 그런데 가희가 다니는 체크 포인트가 수십 개나 되다 보니 조사하기가 몹시 귀찮았다. 가희가 조사하려는 어느 한 구간의 시작 위치와 끝 위치가 주어졌을 때, 그 구간의 평균 기울기를 구하는 프로그램을 작성해서 가희를 도와주자!

## 입력

첫째 줄에 체크 포인트의 개수 *N*(1 ≤ *N* ≤ 50)이 주어진다.

둘째 줄에 시작점의 위치 *X*0와 높이 *H*0가 공백으로 구분되어 주어지며, 시작점의 위치는 항상 0으로 주어진다.

셋째 줄부터 *N*+2번째 줄까지 각 줄에는 체크 포인트 *i*의 위치 *Xi*와 높이 *Hi*가 공백으로 구분되어 주어진다. 시작점을 포함하여 주어지는 위치와 높이는 모두 0 이상 5,000 이하의 정수이며, 항상 *Xi*-1 < *Xi* (단, *i* ≠ 0)를 만족한다.

*N*+3번째 줄에는 가희가 조사하려는 구간의 시작 위치 *S*, 끝 위치 *E*가 주어진다. (*X*0 ≤ *S* < *E* ≤ *X*n)

## 출력

가희가 조사하려는 구간의 평균 기울기의 절댓값을 출력하라. 만약 이 값이 정수라면 정수 값을 출력하고, 그렇지 않다면 기약분수 `p/q`의 형태로 출력하라.
