## 문제

По заданным числам $a$, $b$, $c$ и $d$, найдите наименьшее натуральное число $n$, большее $ac$, которое нельзя представить в виде произведения двух натуральных чисел $u$ и $v$, таких что $a \le u \le b$ и $c \le v \le d$.

## 입력

Входной файл содержит одну строку, состоящую из натуральных чисел $a$, $b$, $c$, $d$ ($1 \le a \le b \le 10^6$, $1 \le c \le d \le 10^6$).

## 출력

Выведите в выходной файл искомое число $n$.
