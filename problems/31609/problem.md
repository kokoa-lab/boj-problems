## 문제

0 以上 9 以下の整数からなる，長さ N の数列 A = (A1, A2, …, AN) が与えられる．数列 A に少なくとも 1 回現れている整数をすべて，小さい順に出力せよ．

## 입력

入力は以下の形式で与えられる．

```

N
A1   A2   …   AN
```

## 출력

数列 A に少なくとも 1 回現れている整数をすべて，小さい順に改行区切りで出力せよ．

結果以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
