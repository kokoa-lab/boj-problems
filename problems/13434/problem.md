## 문제

두 정수 A와 B가 주어진다. 두 정수를 이용해서 평면 위의 A\*B개의 직선을 만들 수 있다.

0 ≤ a < A, 0 ≤ b < B인 모든 (a, b) 쌍은 직선의 방정식 y = ax+b를 만들 수 있다.

A와 B가 주어졌을 때, 평면이 몇 개의 영역으로 나누어지는지 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 A와 B가 주어진다. (1 ≤ A, B ≤ 1,200)

## 출력

첫째 줄에 영역의 개수를 출력한다. 이 값은 int 범위를 넘어간다.
