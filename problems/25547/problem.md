## 문제

두 양의 정수 $A$, $B$가 주어질 때, 다음과 같은 조건을 만족하는 양의 정수 $C$의 개수를 구하여라.

$GCD(A,B) = GCD(A,C)$, $LCM(A,B) = LCM(B,C)$

$GCD(A,B)$는 $A$와 $B$의 최대공약수를, $LCM(A,B)$는 $A$와 $B$의 최소공배수를 의미한다.

## 입력

양의 정수 $A$, $B$가 주어진다. $(1 \leq A,B \leq 10^{9})$

## 출력

조건을 만족하는 양의 정수 $C$의 개수를 출력한다.
