## 문제

正整数 N，A，B が与えられる．ここで，A ≠ B である．

1 以上 N 以下の整数のうち，A と B のいずれか片方のみで割り切れるものの個数を出力せよ．

## 입력

入力は以下の形式で与えられる．

```

N
A
B
```

## 출력

1 以上 N 以下の整数のうち，A と B のいずれか片方のみで割り切れるものの個数を単位 (個) を除いて出力せよ．

答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
