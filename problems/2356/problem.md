## 문제

어떠한 정수 $a$에 대해서도 $a^2$의 십진수 표현이 양의 정수 $N$의 십진수 표현의 부분문자열이 아닐 때 $N$을 **제곱ㄷㄷ수**라고 한다.

제곱ㄷㄷ수를 오름차순으로 나열하면 $2, 3, 5, 6, 7, 8, 22, 23, 26, ...$이 된다.

양의 정수 $N$이 주어질 때 $N$ 이상의 가장 작은 제곱ㄷㄷ수를 구하자.

## 입력

양의 정수 $N$($1 \le N \le 10^{18}$)이 주어진다.

## 출력

$N$ 이상의 가장 작은 제곱ㄷㄷ수를 출력한다.
