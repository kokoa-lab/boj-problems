## 문제

특정 숫자로 이루어진 X의 배수 중에서 A보다 크거나 같고, B보다 작거나 같은것의 개수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 X, A, B가 주어진다. (1 ≤ X ≤ 1011, 1 ≤ A ≤ B ≤ 1011)

둘째 줄에는 사용할 수 있는 숫자가 오름차순으로 중복없이 주어진다.

## 출력

첫째 줄에 A보다 크거나 같고 B보다 작거나 같은 X의 배수 중에서, 입력으로 주어진 숫자로만 이루어진 것의 개수를 출력한다.
