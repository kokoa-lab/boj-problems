## 문제

문자열 S와 정수 K가 주어졌을 때, 다음 쿼리를 수행하는 프로그램을 작성하시오.

* `1 l r c`: S[l..r]을 문자 c로 바꾼다.
* `2 l r`: l ≤ i ≤ j ≤ r이면서, j-i+1 ≤ K를 만족하는 (i, j) 쌍 중에서, S[i..j]가 팰린드롬인 것의 개수를 출력한다.

문자열의 첫 번째 글자의 인덱스는 1이다.

S[i..j]는 S의 i번째 글자부터 j번째 글자까지 부분 문자열을 의미한다.

## 입력

첫째 줄에 문자열 S와 K (1 ≤ K ≤ 50)가 주어진다. S의 길이는 105를 넘지 않으며, 알파벳 소문자로만 이루어져 있다.

둘째 줄에는 쿼리의 개수 M (1 ≤ M ≤ 105)이 주어진다.

셋째 줄부터 M개의 줄에는 쿼리가 한 줄에 하나씩 주어진다.

## 출력

2번 쿼리 마다 한 줄에 하나씩 쿼리의 결과를 순서대로 출력한다.
