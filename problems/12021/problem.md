## 문제

희현이는 학교를 가다가 보물 지도를 주웠다. 보물 지도에 써져 있는 내용은 다음과 같았다.

1. 연세대학교 정문의 좌표를 (0, 0)이라 하자.
2. \(x\_0 = a\), \(y\_0 = b\)이다. (\(0 < b \le a\))
3. \(x\_{n+1} = \frac{(x\_n + y\_n)}{2}\), \(y\_{n+1} = 2\frac{x\_ny\_n}{x\_n+y\_n}\)
4. 계속 쫓아가다 보면 보물은 나오기 마련!

희현이를 도와 보물이 있는 위치를 구하자.

## 입력

첫째 줄에 양의 정수 \(a\)와 \(b\)가 주어진다. (\(0 < b \le a \le 10^{18}\))

## 출력

첫째 줄에 보물이 있는 좌표를 출력하자. 상대/절대 오차는 10-6까지 허용한다.
