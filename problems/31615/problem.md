## 문제

2 つの整数 A,B が与えられる．

A と B の和は十進法で表したとき何桁であるか求めよ．

## 입력

入力は以下の形式で与えられる．

```

A
B
```

## 출력

A と B の和は十進法で表したとき何桁であるか，単位 (桁) を省いて出力せよ．

答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
