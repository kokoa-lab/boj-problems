## 문제

* 길이 $N$의 **순열(permutation)**이란, $1$부터 $N$까지의 수가 각각 한 번씩 등장하는 수열을 의미한다.
* **증가하는 수열(increasing sequence)**이란, 각 원소가 그 앞의 원소보다 큰 수열을 의미한다.
* 어떤 수열의 **부분 수열(subsequence)**이란, 원래 수열에서 몇 개의 원소를 제거하여 얻을 수 있는 수열을 의미한다. 예를 들어, 수열 $[1,3,5]$의 부분 수열로 가능한 것을 모두 나열하면 $[]$, $[1]$, $[3]$, $[5]$, $[1,3]$, $[3,5]$, $[1,5]$, $[1,3,5]$와 같다.
* 어떤 수열의 **최장 증가 부분 수열(longest increasing subsequence, LIS)**이란, 증가하는 부분 수열 중 길이가 최대인 것을 의미한다. 예를 들어, 수열 $[4,1,3,5,2,6]$의 LIS로는 $[1,3,5,6]$이 있으며 길이는 $4$이다.

길이 $N$의 순열 $A$가 주어진다. 이 순열에서 서로 다른 두 위치의 원소를 정확히 한 번 교환하려 한다.

교환 이후의 순열에서 최장 증가 부분 수열(LIS)의 길이가 기존 순열의 LIS의 길이보다 **감소하지 않도록** 할 수 있는지 판별하고, 가능하다면 그러한 두 원소의 위치를 하나 출력하여라.

## 입력

첫째 줄에 정수 $N$이 주어진다. ($2\le N\le 1\, 000\, 000$)

둘째 줄에 서로 다른 $N$개의 정수 $A\_1,A\_2,\cdots ,A\_N$이 공백으로 구분되어 주어진다. ($1\le A\_i\le N$)

## 출력

두 원소를 정확히 한 번 교환하여 LIS의 길이가 감소하지 않도록 만들 수 있다면 첫째 줄에 `YES`를 출력한다. 이 경우 둘째 줄에 교환할 두 원소의 인덱스 $i$, $j$를 공백으로 구분하여 오름차순으로 출력한다. ($1\le i<j\le N$)

그러한 교환이 존재하지 않으면 첫째 줄에 `NO`를 출력한다.

가능한 답이 여러 가지이면 그중 하나만 출력한다.
