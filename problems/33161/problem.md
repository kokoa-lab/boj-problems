## 문제

JOI 君の所持金は A 円である．JOI 君は 1 本 5 円の鉛筆を最大で何本買えるかを求めよ．

## 입력

入力は以下の形式で与えられる．

```

A
```

## 출력

JOI 君が鉛筆を最大で何本買えるか，単位 (本) を除いて出力せよ．

答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
