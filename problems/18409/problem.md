## 문제

長さ N の英小文字からなる文字列 S が与えられる．S のうち母音字の個数，つまり `a`，`i`，`u`，`e`，`o` の個数の総和を求めよ．

## 입력

入力は以下の形式で標準入力から与えられる．

```

N
S
```

## 출력

S のうち母音字の個数，つまり `a`，`i`，`u`，`e`，`o` の個数の総和を出力せよ．
