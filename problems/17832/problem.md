## 문제

준용이와 성현이는 RGB 젠가 게임을 하고 있다. 이는 보통 젠가 게임처럼 블록 뭉치에서 서로 번갈아 가며 블록을 빼며, 블록 뭉치를 무너뜨리는 사람이 패배한다.

RGB 젠가는 3가지 종류의 블록을 사용하며, 블록 아래 붙어있는 스티커의 색으로 이를 구분한다. 세 종류의 블록은 각각 무게가 다르며 블록을 뽑기 전까지는 육안으로 구분할 수 없다.

편의상 빨간색, 초록색, 파란색 스티커가 붙은 각 블록을 R, G, B 라고 칭하고, 각 무게를 *WR*, *WG*, *WB* 라고 한다.

RGB 젠가는 특이하게도 초기 상태에서 *N*만큼의 무게가 줄어들면 곧바로 무너지며, 그전에는 절대 무너지지 않는다.

예를 들어 *N*= 20이고 *WR*= 3, *WG*= 5, *WB*= 7이라고 할 때, R -> G -> G -> B 순서로 블록을 빼면 마지막 B블록을 빼는 순간 블록 뭉치가 무너지며, 해당 블록을 뺀 사람이 패배한다.

준용이가 먼저 블록을 뽑는 것으로 시작한다고 할 때, 누가 이길 확률이 더 높은지 맞혀보자.

이때, 블록 뭉치에는 무수히 많은 블록이 균등하게 분포되어 있어서 R, G, B 각각의 블록을 뽑을 확률은 항상 완벽하게 똑같다고 가정한다.

## 입력

첫 번째 줄에 자연수 *N*이 주어진다. 이때, *N*은 20 이하의 자연수이다.

두 번째 줄에 공백을 사이에 두고 *N*보다 작은 자연수 *WR*, *WG*, *WB*가 주어진다. 이때, 셋은 모두 다른 수이다.

## 출력

준용이가 이길 확률이 높다면 "JUN", 성현이가 이길 확률이 높다면 "SUNG", 확률이 똑같다면 "EQUAL"을 첫째 줄에 출력하라.
