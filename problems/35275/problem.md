## 문제

하늘이네 부대에는 $N$개의 생활관이 있으며, $i$번 생활관에는 $A\_i$명의 인원이 거주하고 있다.

설 연휴를 맞이하여 부대에 보급된 특식 $M$개를 각 생활관에 나눠주려고 한다. 특식은 생활관 단위로 분배되며, 각 생활관에 정수 개수만큼만 줄 수 있다.

생활관 내에서는 모든 사람이 공평하게 특식 양을 나누어 갖는다. 즉, $i$번 생활관에 $B\_i$개의 특식이 분배되었다면 $i$번 생활관의 모든 인원은 $\frac{B\_i}{A\_i}$만큼의 특식을 먹게 된다.이때, 각 사람이 받는 특식의 양이 너무 차이가 나면 분쟁이 생겨날 수 있기 때문에 사람 별로 받는 특식의 양의 분산을 최소화하려고 한다.

$M$개의 특식을 남김없이 생활관에 분배하려고 할 때, 가능한 인당 특식 분배량의 분산의 최솟값을 구하여라.

이때 생활관 단위가 아닌 개인 단위의 인당 특식 분배량에 대해 분산을 구하는 것임에 유의하여라.

## 입력

첫번째 줄에 $N$, $M$이 공백으로 구분되어 주어진다. ($1 \leq N \leq 100,000$, $1 \leq M \leq 1,000,000$)

두번째 줄에 $A\_1$, $A\_2$, $\cdots$, $A\_N$이 공백으로 구분되어 주어진다. ($1 \leq A\_i \leq 10$)

## 출력

각 사람이 먹는 특식의 양의 분산의 최솟값을 출력하여라.

정답과의 절대 오차 또는 상대 오차가 $10^{-9}$ 이하이면 정답으로 처리된다.

## 힌트

분산은 통계학에서 값들이 평균에서 얼마나 떨어져 있는지를 나타내는 수치로, 편차의 제곱의 평균으로 정의된다. 이는 데이터의 흩어진 정도를 측정하며, 분산이 클수록 데이터가 넓게 퍼져 있음을 의미한다.

구체적으로는 $n$개의 데이터 $X\_1$, $X\_2$, $\cdots$, $X\_n$이 있다고 할 때, 평균 $\bar{X} = \frac{1}{n} \sum X\_i$이고 분산 $\text{Var}(X)$은 다음과 같다.

$$ \text{Var}(X) = \frac{1}{n} \sum \left(X\_i - \bar{X}\right)^2 $$
