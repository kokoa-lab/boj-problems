## 문제

수열 s = s1, s2, ..., sn의 점수는 Σi·si로 구할 수 있다.

수열 s의 연속된 부분 수열의 점수의 최댓값을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 n (1 ≤ n ≤ 200,000)이 주어진다.

둘째 줄에는 s1, s2, ..., sn이 주어진다. (|si| ≤ 107)

## 출력

첫째 줄에 수열 s의 연속된 부분 수열의 점수 중 최댓값을 출력한다.
