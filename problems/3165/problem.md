## 문제

N과 K가 주어졌을 때, N보다 크면서 5가 적어도 K번 포함되는 가장 작은 수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 두 자연수 N과 K가 주어진다. (1 ≤ N ≤ 1015, 1 ≤ K ≤ 15)

## 출력

첫째 줄에 문제의 정답을 출력한다.
