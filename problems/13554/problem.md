## 문제

길이가 N인 두 수열 A1, A2, ..., AN과 B1, B2, ..., BN가 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.

* `i j k`: i ≤ p, q ≤ j이면서 Ap × Bq ≤ k 인 (p, q) 쌍의 개수를 출력한다.

## 입력

첫째 줄에 수열의 크기 N (1 ≤ N ≤ 100,000)이 주어진다.

둘째 줄에는 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 100,000)

둘째 줄에는 B1, B2, ..., BN이 주어진다. (1 ≤ Bi ≤ 100,000)

넷째 줄에는 쿼리의 개수 M (1 ≤ M ≤ 100,000)이 주어진다.

다섯째 줄부터 M개의 줄에는 쿼리 i, j, k가 한 줄에 하나씩 주어진다. (1 ≤ i ≤ j ≤ N, 1 ≤ k ≤ 100,000)

## 출력

각각의 쿼리마다 정답을 한 줄에 하나씩 출력한다.
