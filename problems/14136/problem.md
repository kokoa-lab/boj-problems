## 문제

Zadan je niz prirodnih brojeva. Pronañite duljinu najduljeg rastućeg podniza u kojem je svaki broj višekratnik prethodnog broja.

## 입력

U prvom redu nalazi se cijeli broj N (1 ≤ N ≤ 100000), duljina niza.

U svakom od sljedećih N redova nalazi se po jedan prirodni broj manji od 1000000, redom brojevi promatranog niza.

## 출력

U prvi i jedini red ispišite duljinu najduljeg rastućeg podniza u kojem je svaki broj višekratnik prethodnog broja.
