## 문제

와칠과 토카는 소인수분해 게임을 하려 한다. 소인수분해 게임의 규칙은 다음과 같다.

* 처음에 칠판에 양의 정수 $N$이 하나 적혀 있는 상태로 시작한다.
* 두 플레이어는 번갈아가며 칠판에 적힌 수의 임의의 소인수를 하나 선택해 칠판에 적힌 수를 해당 소인수로 나눈 몫으로 변경한다. 이때 상대방이 직전에 선택한 소인수는 선택할 수 없다.
  + 단, 선공의 첫 번째 차례에는 상대방이 직전에 선택한 소인수가 없으므로 아무 소인수나 선택해도 된다.
* 선택할 수 있는 소인수가 없다면 해당 플레이어가 패배하고 상대방이 승리하게 된다.

각 게임에서 처음에 칠판에 적힌 수 $N$이 주어질 때마다 와칠과 토카가 모두 최선을 다해서 플레이 한다면 누가 승리할 지 맞혀 보자. 게임은 항상 와칠부터 시작한다.

## 입력

첫째 줄에 와칠과 토카가 진행할 게임의 수 $T$ $(1 \le T \le 1\,000)$가 주어진다.

이후 $T$개의 줄에 걸쳐 각 게임에서 처음에 칠판에 적혀있는 정수 $N$ $(2 \le N \lt 10^{12})$이 주어진다.

## 출력

각 게임마다 와칠이 승리한다면 `yyyy7089`, 토카가 승리한다면 `toycartoon`을 한 줄에 하나씩 출력한다.
