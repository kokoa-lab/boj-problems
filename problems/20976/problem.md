## 문제

3 つの整数 A, B, C が与えられる．これらのうち 2 番目に大きい数を出力せよ．

## 입력

入力は以下の形式で標準入力から与えられる．

A B C

## 출력

A, B, C のうち，2 番目に大きい数を出力せよ．
