## 문제

채점 프로그램과 가위 바위 보를 1,002번 해서, 최대한 많이 이겨 보자.

채점 프로그램이 가위 바위 보를 내는 순서는 미리 결정되어져 있으며, 이 순서는 유저마다 서로 다르게 결정되어져 있다.

## 입력

첫째 줄에 길이가 N (1 ≤ N ≤ 1,002)인 문자열 S가 주어진다. N은 채점 순서를 의미하고, S는 '`1`'로만 이루어져 있다.

항상 N = 1인 데이터를 가장 먼저 채점하게 되고, 그 다음은 N = 2, N = 3, ..., N = 1,002의 순서로 채점한다.

## 출력

첫째 줄에 무엇을 낼 것인지 출력한다.

가위는 0, 바위는 1, 보는 2이다.
