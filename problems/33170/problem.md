## 문제

3 つの整数 A, B, C が与えられる．

3 つの値の合計が 21 以下なら 1 を，そうでないなら 0 を出力せよ．

## 입력

入力は以下の形式で与えられる．

```

A
B
C
```

## 출력

3 つの値の合計が 21 以下なら 1 を，そうでないなら 0 を出力せよ．

答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)
