## 문제

두 자연수 *N,* *K*가 입력되었을 때 *N*을 *K*개의 소수의 합으로 분해하는 프로그램을 작성하시오.

## 입력

첫째 줄에 자연수 *N*(1 ≤ *N* ≤ 100,000,000), *K*(1 ≤ *K* ≤ 10,000)이 주어진다.

## 출력

첫째 줄에 *K*개의 소수를 빈 칸을 사이에 두고 순서대로 출력한다. *K*개의 소수가 정렬되어 있지 않아도 된다. 불가능한 경우에는 -1을 출력한다.
