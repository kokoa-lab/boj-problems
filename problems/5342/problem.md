## 문제

The accounting office is having troubling with billing each department for the supplies purchased last semester. Below is a chart of all the supplies and how much they cost. You must write a program that reads a list of the supplies and computes the total cost.

| Item | Cost |
| --- | --- |
| Paper | 57.99 |
| Printer | 120.50 |
| Planners | 31.25 |
| Binders | 22.50 |
| Calendar | 10.95 |
| Notebooks | 11.20 |
| Ink | 66.95 |

## 입력

The input is a list of office supplies with one item per line. End-of- input is denoted by the string “EOI”.

## 출력

Print the total cost of all the items in the list. The output must start with a dollar sign, “$” and the total printed with exactly two decimal digits.
