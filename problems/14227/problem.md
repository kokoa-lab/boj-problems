## 문제

영선이는 두 정수 a와 b를 가지고 있다.

효빈이는 빨간 버튼과 파란 버튼을 가지고 있다.

빨간 버튼을 누르면 영선이가 가지고 있는 두 정수의 값이 1 증가한다. 파란 버튼을 누르면 두 정수의 값에 2를 곱한다.

효빈이는 (a, b)를 (c, d)로 바꾸려고 한다. 이때 버튼을 누르는 횟수의 최솟값을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 a, b, c, d가 주어진다. (1 ≤ a, b, c, d ≤ 1,000,000,000)

## 출력

첫째 줄에 (a, b)를 (c, d)로 바꾸는데 필요한 연산 횟수의 최솟값을 출력한다. 만약, 불가능한 경우에는 -1을 출력한다.
