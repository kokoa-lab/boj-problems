## 문제

구구단을 외우던 호반우는 $2\,000\,000$ 이하의 소수가 아닌 양의 정수 $N$개를 골라 공차가 $M$인 등차수열을 만들 수 있을지 궁금해졌다.

아쉽게도 멀티태스킹에 약한 호반우를 위해 등차수열을 찾아주자.

## 입력

첫째 줄에 양의 정수 $N$과 $M$이 공백을 두고 주어진다. $(2 \leq N, M \leq 1\,000)$

## 출력

만약 조건을 만족하는 등차수열을 만들 수 없다면 첫째 줄에 `-1`을 출력한다.

그렇지 않다면 첫째 줄에 길이가 $N$이고 공차가 $M$인 등차수열을 순서대로 공백을 두고 출력한다.

가능한 방법이 여러 가지라면 그중 아무거나 출력한다.
