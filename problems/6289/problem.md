## 문제

뒤에서부터 읽어도 원래 문자열과 같은 문자열을 팰린드롬이라고 한다. 숫자 A의 부분 문자열 중에서 길이가 1보다 큰 팰린드롬이 없는 숫자를 무-팰린드롬 숫자라고 한다.

16276는 무-팰린드롬 숫자이다. 하지만, 17276의 부분 문자열 727은 팰린드롬이기 때문에 17276은 무-팰린드롬 숫자가 아니다.

두 수 a와 b가 주어졌을 때, a보다 크거나 같고, b보다 작거나 같은 숫자 중에서 무-팰린드롬 숫자의 개수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 두 정수 a와 b가 주어진다. (0 ≤ a ≤ b ≤ 1018)

## 출력

a보다 크거나 같고, b보다 작거나 같은 정수 중에서 무-팰린드롬 숫자의 개수를 첫째 줄에 출력한다.
