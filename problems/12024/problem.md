## 문제

주어진 그래프가 있다. 문제는 간단하다 사각형의 개수를 찾는 것이다. 즉, 서로 다른 정점들로 이루어진 길이가 4인 Cycle의 개수를 세는 것이다. 이때 정점 방문 순서가 다르면 다른 경우로 간주한다. (1 -> 2 -> 3 -> 4 -> 1 과 2 -> 3 -> 4 -> 1 -> 2은 다른 경우이다.)

## 입력

정점의 개수 N이 주어진다. (1 ≤ N ≤ 250)

그 후 N줄에는 그래프가 양방향 그래프인 인접행렬 A로 주어진다. Aii가 1인 경우는 없다.

## 출력

서로 다른 길이 4짜리의 Cycle의 개수를 출력한다.
