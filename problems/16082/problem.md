## 문제

Хоёр ижил цифр дараалан орсон тоог муухай тоо, бусад тоонуудыг гоё тоо гэж нэрлэе. n бүхэл тоо өгөгдөв. n – ээс эрс их байх бөгөөд хамгийн бага гоё тоог ол.

## 입력

n бүхэл тоо байрлана (1 ≤ n ≤ 1018).

## 출력

Дараалсан хоёр ижил цифр агуулаагүй, n – ээс эрс их байх хамгийн бага бүхэл тоог гаргана.
