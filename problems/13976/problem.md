## 문제

3×N 크기의 벽을 2×1, 1×2 크기의 타일로 채우는 경우의 수를 구해보자.

## 입력

첫째 줄에 N(1 ≤ N ≤ 1,000,000,000,000,000,000)이 주어진다.

## 출력

첫째 줄에 경우의 수를 1,000,000,007로 나눈 나머지를 출력한다.
