## 문제

Senya likes integers not containing two consecutive equal digits. You are given an integer $n$. Help Senya to find the smallest integer he likes, which is strictly greater than $n$.

## 입력

Input contains integer $n$ ($1 \le n \le 10^{18}$).

## 출력

Print the smallest integer strictly greater than $n$, which doesn't contain any two consecutive equal digits.
