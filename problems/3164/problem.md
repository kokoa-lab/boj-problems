## 문제

상근이는 제1사분면에 아래와 같은 패턴을 그리고 있다.

![](./001_preview)

직사각형이 주어졌을 때, 직사각형에 포함되는 어두운 정사각형의 수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 직사각형의 왼쪽 아래 꼭짓점 좌표(X1, Y1)와 오른쪽 위 좌표(X2, Y2)가 주어진다. (0  ≤ X1 < X2 ≤ 1,000,000, 0 ≤ Y1 < Y2 ≤ 1,000,000)

## 출력

입력으로 주어진 직사각형에 포함된 어두운 정사각형의 수를 출력한다.
