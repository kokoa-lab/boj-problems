## 문제

선린이는 효율적으로 코딩하는 걸 좋아해서 아래와 같은 방식으로 자료형을 선택한다고 한다. 선린이가 선택할 자료형을 알아내자!

**선린이가 자료형을 선택하는 기준은 다음과 같다.**

* 정확하게 표현할 수 있으면서, 메모리를 적게 차지하는 자료형을 선택한다.
* 선택할 수 있는 자료형은 `short`, `int`, `long long` 중 하나이다.

|  |  |  |
| --- | --- | --- |
| 자료형 | 최솟값 | 최댓값 |
| `short` | $-2^{15}$  $-32,768$ | $2^{15}-1$  $32,767$ |
| `int` | $-2^{31}$  $-2,147,483,648$ | $2^{31}-1$  $2,147,483,647$ |
| `long long` | $-2^{63}$  $-9,223,372,036,854,775,808$ | $2^{63}-1$  $9,223,372,036,854,775,807$ |

## 입력

정수 $N$이 주어진다.

## 출력

선린이가 선택할 자료형을 출력한다.
