## 문제

2차원 평면위에 있는 점 N개가 주어진다. 모든 점을 포함하는 가장 작은 원의 지름을 구하는 프로그램을 작성하시오. 원의 둘레 위에 있는 점도 포함하는 점이다.

## 입력

첫째 줄에 점 N (2 ≤ N ≤ 300)이 주어진다. 둘째 줄부터 N개의 줄에 점의 좌표 x, y가 주어진다. (0 ≤ x, y ≤ 1,000)

## 출력

모든 점을 포함하는 원의 지름의 최솟값을 반올림해서 소수점 둘째자리까지 출력한다.
