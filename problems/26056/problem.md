## 문제

양의 정수 $n$의 모든 양의 약수를 $d\_1,d\_2,\cdots ,d\_k$라 할 때, $a\_n=\left( -1 \right)^{d\_1}+\left( -1 \right)^{d\_2}+\cdots +\left( -1 \right)^{d\_k}$으로 정의한다.

양의 정수 $S,T$가 주어질 때, $\sum\_{i=S}^{T}{a\_i}$을 구하시오.

## 입력

첫 번째 줄에 양의 정수 $S,T$가 공백으로 구분되어 주어진다. $(1\le S\le T\le 10^{14})$

## 출력

$\sum\_{i=S}^{T}{a\_i}$을 출력한다.
