## 문제

JOI 高校には 2N (= 2 ✕ N) 人の生徒がおり，1 から 2N までの番号が付けられている．

来月 JOI 高校では運動会が開催され，その中の競技「二人三脚」では，2N 人の生徒が N 組の 2 人組に分かれレースを行う．

組には 1 から N までの番号が付けられており，生徒 i (1 ≦ i ≦ 2N) は組 Ai に属している．

あなたはどの生徒がどの組に属しているかの表を作ったが，どういうわけか，生徒 2N がどの組に属しているか分からなくなってしまった．

A1, A2, …, A2N-1 が与えられるので，生徒 2N が属する組の番号 A2N を求めよ．

## 입력

入力は以下の形式で標準入力から与えられる．

```

N
A1 A2 … A2N-1
```

## 출력

生徒 2N が属する組の番号 A2N を出力せよ．
