## 문제

長さ N の整数列 A = (A1, A2, …, AN) が与えられる．

A に出現する整数のうち，出現回数が最小である整数を出力せよ．ただし，そのような整数が複数考えられる場合は，考えられる整数のうち最も小さい整数を出力せよ．

## 입력

入力は以下の形式で標準入力から与えられる．

```

N
A1 A2 … AN
```

## 출력

A に出現する整数のうち，出現回数が最小である整数を出力せよ．ただし，そのような整数が複数考えられる場合は，考えられる整数のうち最も小さい整数を出力せよ．
