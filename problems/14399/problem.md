## 문제

1이 저장되어 있는 두 변수 X와 Y가 있다.

사용할 수 있는 연산은 아래와 같다.

* 연산 X: X = X+Y
* 연산 Y: Y = X+Y

N이 주어진다. 두 연산을 사용해서 두 변수 중 하나에 N을 저장하는 프로그램을 작성하시오. 가능한 연산 방법이 여러 가지면, 연산의 길이가 가장 짧은 것을, 그러한 것도 여러 가지면 사전 순으로 앞서는 것을 출력한다.

## 입력

첫째 줄에 N (1 ≤ N ≤ 1,000,000)이 주어진다.

## 출력

첫째 줄에 연산 방법을 출력한다.
