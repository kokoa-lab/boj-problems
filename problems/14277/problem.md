## 문제

등차 수열 A의 일반항 a + b\*x (x는 음이 아닌 정수)에서 a, b와 등비 수열 G의 일반항 c \* d^y (y는 음이 아닌 정수)에서 c, d가 주어진다.

u가 주어졌을 때, 1부터 u까지의 수 중에서 입력으로 주어진 등차 수열 A나 등비 수열 G에 포함되는 수의 개수를 구하는 프로그램을 작성하시오. (동시에 포함되어도 된다)

## 입력

첫째 줄에 a, b, c, d, u가 주어진다. (1 ≤ a, b, c, u ≤ 1012, 1 ≤ d ≤ 105)

## 출력

첫째 줄에 등차 수열 A나 등비 수열 G에 포함되는 1보다 크거나 같고 u보다 작거나 같은 자연수의 개수를 출력한다.

## 힌트

예제 4의 경우에 4, 20, 100, 452, 476, 500, 524, 548, 572, 596이 정답이다.
