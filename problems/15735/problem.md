## 문제

N층의 정삼각형이 주어졌을때, 크고 작은 정삼각형의 개수가 몇개인지 알아보자.

N이 2일 경우 정답은 5이다.

![](./001_1.png)

## 입력

첫 번째 줄에 정삼각형 탑의 층 N이 주어진다. (1 ≤ N ≤ 10,000)

## 출력

첫 번째 줄에 크고 작은 삼각형의 개수의 합을 출력한다.
