## 문제

Изобретенные перед предыдущим отборочным раундом трехцветные шахматы очень понравились всем членам жюри, и они решили продолжить эксперименты с этой популярной игрой. Теперь один из членов жюри предложил увеличить количество ладей каждого из игроков до трех. Чтобы понять, насколько интересна будет игра по новым правилам, члены жюри решили посмотреть, сколько клеток будет находиться «под боем» при различных расположениях ладей.

Клетка находится под боем если в этой клетке нет ладьи, и существует такая ладья, которая находится на одной вертикали или горизонтали с этой клеткой.

Для анализа игры жюри хочет понять, возможно ли расставить на доске три ладьи так, чтобы под боем находилось ровно *k* клеток? Если это возможно, необходимо найти такое расположение.

## 입력

Первая строка содержит целое число *T* (1 ≤ *T* ≤ 104) — количество тестовых примеров. В каждой из следующих *T* строк содержится по три целых неотрицательных числа *n*, *m* и *k*, где *n* и *m* – размеры шахматного поля (1 ≤ *n, m* ≤ 109, 0 ≤ *k* ≤ 109).

## 출력

Для каждого из *T* тестовых примеров выведите ответ в одной строке. Если расставить ладьи искомым образом ладьи нельзя, то выведите «IMPOSSIBLE». Иначе выведите три пары чисел – координаты ладей. Первое число должно быть в диапазоне от 1 до *n*, а второе — от 1 до *m*. Две ладьи нельзя ставить в одну клетку.

В этой задаче вместо WA в первые 30 минут контеста мог выдаваться PE. Приносим свои извинения.
