## 문제

В математике часто встречаются так называемые *рекуррентные соотношения*. Обычно они применяются для задания числовых последовательностей --- очередное число в последовательности некоторым образом выражается через предыдущие. Примером такой последовательности являются числа Фибоначчи (в них очередное число равно сумме двух предыдущих).

С помощью соотношений такого типа можно задавать не только последовательности чисел, но и последовательности строк. В этой задаче рассматривается последовательность строк $s\_0$, $s\_1$, \ldots, задаваемая следующим образом.

Строка $s\_0$ пуста, а каждая строка $s\_i$ ($i \ge 1$) получается из $s\_{i-1}$ по следующему правилу: если десятичная запись числа $i$ входит как подстрока в $s\_{i-1}$, то $s\_i = s\_{i-1}$, иначе $s\_i$ получается приписыванием к $s\_{i-1}$ в конец десятичной записи числа $i$.

Задано число $n$. Необходимо найти строку $s\_n$.

## 입력

Входной файл содержит целое число $n$ ($1 \le n \le 500$).

## 출력

В выходной файл выведите строку $s\_n$.
