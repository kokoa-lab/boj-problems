## 문제

이 문제는 가위 바위 보를 하는 문제입니다.

가위 바위 보 문제는 시즌제로 운영되며, 각 시즌 준비 기간에 여기에 제출을 하면 참가 신청이 완료됩니다.

각 시즌의 규칙은 시즌마다 다를 수도 있기 때문에, 가위 바위 보 페이지를 참고해주세요.

[https://www.acmicpc.net/rps](./001_rps)

내 프로그램이 상대방과 총 N번 경기를 한 결과가 입력으로 주어진다. 이때, 내가 다음에 무엇을 낼지 출력하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N이 주어진다. (0 ≤ N ≤ 10,000)

둘째 줄부터 N개의 줄에 내가 낸 것과 상대방이 낸 것이 입력으로 주어진다.

가위는 0, 바위는 1, 보는 2이다.

## 출력

내가 무엇을 낼지 출력한다.

가위는 0, 바위는 1, 보는 2이다.

## 힌트

예제 1~4는 순서대로 경기가 어떤식으로 진행되는지를 나타냅니다.

예제 1은 경기의 시작이므로, 아직 아무 결과도 입력으로 주어지지 않습니다.
