## 문제

**NEMODEMIC**은 [App Store](./001_id6757725643)와 [Google Play Store](./002_details)에서 화제가 되고 있는 2D 퍼즐게임이다. **NEMODEMIC**는 바이러스가 퍼지는 격자 보드에서 플레이어가 **시작점**에서 출발하여 **탈출구**를 향해 이동하는 탈출 게임이다. 사방으로 확산되는 **바이러스**를 피해서 전략적으로 **백신**을 수집하며 **탈출구**에 도달해야 한다.

**NEMODEMIC**은 $N$개의 행과 $M$개의 열로 이루어진 격자 모양의 보드에서 진행된다. 보드의 각 칸은 **빈 칸**, **벽**, **바이러스**, **백신**, **시작점**, **탈출구** 중 하나이다.

![](./003_preview)

**빈 칸**: 플레이어가 이동하거나 바이러스가 확산될 수 있는 칸이다.

**벽**: 플레이어가 뚫고 지나가거나 위치할 수 없으며, 바이러스 역시 벽을 뚫고 확산될 수 없다.

**바이러스**: 상, 하, 좌, 우 네 방향 중 **활성화된 방향**으로만 확산된다. 바이러스는 **한 방향 바이러스**와 **모든 방향 바이러스**로 나뉜다. **한 방향 바이러스**는 상, 하, 좌, 우 중 한 방향만 **활성화된 방향**이다. **모든 방향 바이러스**는 상, 하, 좌, 우 네 방향 모두 **활성화된 방향**이다. 확산은 해당 방향 혹은 방향들로 매턴 한 칸씩 확산된다.

**백신**: 플레이어가 백신 칸에 도달하면 백신을 수집한다. 백신 하나당 한 개의 바이러스가 확산된 칸을 지나갈 수 있게 한다. 플레이어는 여러 개의 백신을 수집하여 가지고 있을 수 있다. 백신이 있는 칸도 바이러스가 확산될 수 있으며, 이미 바이러스가 확산된 칸에 있는 백신을 수집하기 위해서는 기존에 가지고 있는 백신을 하나 소모해야만 한다.

**시작점**: 플레이어가 시작하는 위치로, 이를 제외하고는 **빈 칸**과 같다.

**탈출구**: 플레이어가 도착해야 하는 위치로, 이를 제외하고는 **빈 칸**과 같다. 탈출구에도 바이러스가 확산될 수 있으며, 바이러스가 확산된 탈출구로 이동하기 위해서는 백신을 하나 소모해야만 한다.

플레이어는 매턴 상, 하, 좌, 우 네 방향 중 벽이 아닌 한 방향으로 한 칸 움직인다. 이때 바이러스가 이미 어떤 칸에 확산되었거나 확산된 턴에는, 백신을 하나 소모해야 해당 칸으로 움직일 수 있다. 플레이어는 보드를 벗어날 수 없다.

바이러스 확산과 관련해 한 가지 예시를 보자. $N=5$, $M=5$이고 $\left( 2,2 \right)$에 벽이 $1$개, $\left( 3,3 \right)$에 우측 방향 **한 방향 바이러스**가 $1$개, $\left( 4,1 \right)$에 백신이 $1$개, $\left( 5,4 \right)$에 **모든 방향 바이러스**가 $1$개 있는 보드를 생각하자. **한 방향 바이러스**는 지정된 방향으로만, **모든 방향 바이러스**는 네 방향 모두로 확산된다.

![](./004_preview)

이 게임은 여러 스테이지로 구성되어 있으며, 각 스테이지는 보드에 존재하는 칸들의 조건에 따라 아래와 같이 $1$단계부터 $4$단계까지의 **난이도** 중 하나를 가진다. 여러 난이도의 조건을 동시에 만족하면 가장 낮은 난이도로 판단된다.

| **난이도** | **빈 칸** | **벽** | **한 방향 바이러스** | **모든 방향 바이러스** | **백신** | **시작점** | **탈출구** |
| --- | --- | --- | --- | --- | --- | --- | --- |
|  | | | | | | | |
| $1$ | 제한 없음 | $1$개 이하 | 방향과 관계없이 총 $1$개 이하 | $0$개 | | 각각 $1$개 | |
| $2$ | 제한 없음 | | |
| $3$ | $0$개 | 제한 없음 |
| $4$ | 제한 없음 | | | | |

게임 올 클리어를 목표로 하는 유림이는 주어지는 스테이지의 보드를 보고 해당 스테이지의 **난이도**를 알고 싶다. 주어진 스테이지의 **난이도**를 알아내 보자.

## 입력

첫 번째 줄에 스테이지 보드의 행과 열의 개수 $N,M(1\le N,M\le 50)$이 공백으로 구분되어 주어진다.

두 번째 줄부터 $N$줄에 걸쳐 보드판의 각 행을 나타내는 길이 $M$의 `.`, `#`, `U`, `D`, `L`, `R`, `A`, `V`, `S`, `E`로만 이루어진 문자열이 공백 없이 주어진다. 각 문자의 의미는 아래와 같다.

* `.` : **빈 칸**
* `#` : **벽**
* `U`, `D`, `L`, `R` : **한 방향 바이러스**로, 각각 **활성화된 방향**이 상, 하, 좌, 우다.
* `A` : **모든 방향 바이러스**
* `V` : **백신**
* `S` : **시작점**
* `E` : **탈출구**

## 출력

첫 번째 줄에 주어진 스테이지의 **난이도**를 출력한다. 만약 조건을 만족하는 난이도가 없다면 대신 `-1`을 출력한다.
