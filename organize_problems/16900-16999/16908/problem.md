## 문제

문자열 N개가 주어졌을 때, 모든 문자열에 등장하는 가장 긴 부분 문자열을 구하시오.

## 입력

첫째 줄에 문자열의 개수 N(2 ≤ N ≤ 10)이 주어진다. 둘째 줄부터 N개의 줄에 문자열이 한 줄에 하나씩 주어진다. 문자열 하나의 길이는 100,000보다 작거나 같고, 알파벳 소문자로만 이루어져 있다.

## 출력

모든 문자열에 등장하는 가장 긴 부분 문자열의 길이를 출력한다.
