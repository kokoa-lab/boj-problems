## 문제

어떤 양의 정수의 제곱인 수를 완전제곱수라고 부른다. 양의 정수 $N$이 주어지면 $N$이 완전제곱수인지 판별하자.

## 입력

첫째 줄에 테스트 케이스의 개수 $T$가 주어진다.

둘째 줄부터 $T$개의 줄에 걸쳐 양의 정수 $N$이 한 줄에 하나씩 주어진다.

## 출력

$T$개의 줄에 걸쳐 각 테스트 케이스마다 $N$이 완전제곱수라면 `1`, 그렇지 않으면 `0`을 출력한다.
