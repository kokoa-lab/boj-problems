## 문제

##### 

`haru_101`이 플레이하는 게임에 **`/gg`** 명령어가 있다. 이 명령어는 항복 투표를 진행하기 위한 명령어로, 해당 팀의 각 팀원은 **`/gg`**를 입력함으로써 항복 의사를 밝힐 수 있다. 팀원 중 절반 이상이 **`/gg`** 명령어를 입력하면 팀은 항복하게 된다.

각 팀원의 명령어 입력 여부가 주어졌을 때, 해당 팀이 항복하게 되는지를 알아보자.

## 입력

첫째 줄에 팀원의 수를 나타내는 정수 $N$이 주어진다.

둘째 줄에 길이가 $N$인 알파벳 대문자 `O`와 `X`로만 이루어진 문자열 $S$가 주어진다.

$S$의 $i$번째 문자는 $i$번째 팀원이 **`/gg`** 명령어를 입력했다면 `O`, **`/gg`** 명령어를 입력하지 않았다면 `X`다.

## 출력

해당 팀이 항복하게 되면 `Yes`, 그렇지 않다면 `No`를 출력한다.

## 힌트

실제 게임에서의 **`/gg`** 명령어와 작동 방식이 다름에 유의하라.
