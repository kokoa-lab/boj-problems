## 문제

길이가 N인 수열 A1, A2, ..., AN이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오

* `1 i v`: Ai를 v로 바꾼다. (1 ≤ i ≤ N, 1 ≤ v ≤ 104)
* `2 i j k`: Ai, Ai+1, ..., Aj로 이루어진 부분 수열 중에서 k보다 큰 원소의 개수를 출력한다. (1 ≤ i ≤ j ≤ N, 1 ≤ k ≤ 104)

수열의 인덱스는 1부터 시작한다.

## 입력

첫째 줄에 수열의 크기 N (1 ≤ N ≤ 100,000)이 주어진다.

둘째 줄에는 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 104)

셋째 줄에는 쿼리의 개수 M (2 ≤ M ≤ 200,000)이 주어진다.

넷째 줄부터 M개의 줄에는 쿼리가 한 줄에 하나씩 주어진다. 2번 쿼리는 하나 이상 주어진다.

## 출력

2번 쿼리에 대해서 정답을 한 줄에 하나씩 순서대로 출력한다.
