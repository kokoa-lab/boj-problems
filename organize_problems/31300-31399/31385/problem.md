## 문제

Android Vasya attends Maths classes. His group started to study the number theory recently. The teacher gave them several tasks as a homework. One of them is as follows.

There is an integer $n$. The problem is to find a sequence of integers $a\_1, \ldots, a\_n$ such that for any $k$ from 2 to $n$ the sum $a\_1 + \ldots + a\_k$ has exactly $a\_k$ different positive divisors. Help Vasya to cope with this task.

## 입력

The only line contains an integer $n$ ($2 \leq n \leq 100\,000$).

## 출력

If there is no such sequence output `Impossible`. Otherwise output space-separated integers $a\_1, \ldots, a\_n$ ($1 \leq a\_i \leq 300$).
