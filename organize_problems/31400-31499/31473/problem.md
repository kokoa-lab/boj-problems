## 문제

시현이의 방구석에 핑크색 머리카락과 하늘색 머리카락이 각각 $N$개씩 떨어져 있었다!

시현이는 이것들이 단비 양의 머리카락이라고 확신했다.

시현이가 열심히 머리카락들의 길이를 잰 결과 $i$번째 핑크색 머리카락의 길이는 $A\_i$였고, $i$번째 하늘색 머리카락의 길이는 $B\_i$였다. $(1 \leq i \leq N)$

열심히 머리카락들의 길이를 재고 나니 시현이는 $\left | a\sum\_{i=1}^N A\_i - b\sum\_{i=1}^N B\_i \right|$를 최소화할 수 있는 $-1\,000\,000$ 이상 $1\,000\,000$ 이하의 $0$이 아닌 정수 $a$와 $b$가 무엇인지 궁금해졌다.

시현이의 궁금증에 대신 답해주자!

## 입력

첫째 줄에 머리카락 쌍의 개수 $N$이 주어진다. $(1 \leq N \leq 1\,000)$

둘째 줄에 $A\_1, A\_2, \cdots, A\_N$, 셋째 줄에 $B\_1, B\_2, \cdots, B\_N$이 공백으로 구분되어 주어진다. $(1 \leq A\_i, B\_i \leq 100)$

입력의 모든 수는 정수이다.

## 출력

첫째 줄에 $\left |a\sum\_{i=1}^N A\_i - b\sum\_{i=1}^N B\_i \right|$가 최소가 되는 $a$와 $b$를 공백으로 구분하여 출력한다.

그러한 $(a, b)$ 쌍이 여러 개라면, 그중 아무거나 출력한다.
