## 문제

長さ N の整数列 A = (A1, A2, ..., AN) と長さ M の整数列 B = (B1, B2, ..., BM) が与えられる．A と B の両方に出現する整数をすべて，1 個ずつ昇順で出力せよ．

## 입력

入力は以下の形式で標準入力から与えられる．

N M  
A1 A2 ... AN  
B1 B2 ... BM

## 출력

A と B の両方に出現する整数をすべて，1 個ずつ昇順で出力せよ．各整数は改行で区切ること．
