## 문제

Jack has $N$ coins, with values $M\_1$, $M\_2$, \dots, $M\_N$. Find the smallest positive amount that cannot be paid with these coins with no change.

## 입력

The first line contains $N$ ($1 \le N \le 1\,000$), the number of coins. The second line contains $N$ integers $M\_i$ ($1 \le M\_i \le 1\,000\,000$), the values of the coins.

## 출력

The only line should contain a single positive integer: the smallest amount that Jack cannot pay with his coins.
