## 문제

В этой задаче упоминается популярная в последнее время игра 2048. Знание ее правил не требуется для решения задачи.

В свободное время Вася любит играть в популярную игру 2048. После того, как Вася догадался до правильной стратегии, ему наскучила эта игра и он решил сделать похожую игру с собственными правилами.

Игровое поле для Васиной игры имеет форму квадрата размером *n* × *n*, в некоторых клетках которого записаны числа. Будем называть игровое поле *хорошим*, если на поле есть свободная клетка, или есть две соседние по стороне клетки, в которых стоят одинаковые числа.

По заданному игровому полю определите, является ли оно хорошим.

## 입력

Входные данные содержат несколько тестовых примеров. Первая строка содержит число *T* — число тестовых примеров (1 ≤ *T* ≤ 100). Далее идет описание самих примеров в следующем формате.

Первая строка описания содержит одно натуральное число *n* (1 ≤ *n* ≤ 10) — размер игрового поля. Следующие *n* строк содержат по *n* неотрицательных целых чисел — описание игрового поля. Пустая клетка задается числом 0, заполненная клетка задается записанным в ней целым положительным числом от 1 до 1000.

## 출력

Выведите *T* строк. Для каждого тестового набора выведите «`YES`», если поле является хорошим, либо «`NO`» в противном случае.
