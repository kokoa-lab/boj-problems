## 문제

어떤 수열을 오름차순으로 정렬했을 때 등차수열이 된다면 그 수열을 **특별한 수열**이라고 한다. 길이가 $1$인 수열은 특별한 수열이다.

길이가 $N$인 수열 $A$가 주어진다. 모든 $1\leq i\leq N$에 대해, $A$의 첫 $i$개의 수로 이루어진 수열이 특별한 수열인지 알아내라.

## 입력

첫째 줄에 수열의 길이 $N$이 주어진다. ($1 \leq N \leq 200\,000$)

둘째 줄에 수열의 원소 $A\_1, A\_2, \ldots, A\_N$이 공백으로 구분되어 주어진다. ($1 \leq A\_i \leq 10^9$)

## 출력

$N$개의 줄에 걸쳐, $i$번째 줄에는 $A$의 첫 $i$개의 수로 이루어진 수열이 특별한 수열이라면 `1`을, 아니라면 `0`을 출력한다.
