## 문제

알고리즘 수업에는 총 $N$개의 강의평들이 달려 있다. $i$번째 강의평의 점수 $A\_i$는 $0$ 이상 $100$ 이하의 정수이다.

알고리즘을 너무나도 좋아하는 로하는 이 수업에 $100$점짜리 강의평을 최소 개수만큼 추가해서 평균 점수를 $X$ 이상으로 만들기로 했다.

$100$점짜리 강의평을 최소 몇 개 추가해야 평균 점수가 $X$ 이상이 될지 구하여라.

## 입력

첫째 줄에 강의평의 개수 $N$과 로하가 목표로 하는 평균 점수 $X$가 주어진다. ($1 \leq N \leq 100\,000$; $0 \leq X \leq 99$)

둘째 줄에 강의평의 점수 $A\_1,A\_2,\cdots,A\_N$이 공백으로 구분되어 주어진다. ($0 \leq A\_i \leq 100$)

## 출력

첫째 줄에 최소로 추가해야 하는 강의평의 개수를 출력한다.
