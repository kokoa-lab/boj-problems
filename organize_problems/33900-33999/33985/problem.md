## 문제

벚꽃컵을 준비하던 준혁이는 지속된 문제 유출로 인해 성준이에게 크게 혼나게 되었다. 준혁이는 유출을 막기 위해 `A`와 `B`로 이루어진 길이 $N$의 암호 $T$를 만들어 성준이에게 보내려고 한다.

$N$개의 문자를 저장할 수 있는 문자열 $S$가 있다. 처음에 $S$의 모든 문자는 `X`이다. 준혁이는 $S$에 다음과 같은 연산을 할 수 있다.

* $(1 \leq i \leq N-1)$인 $i$를 골라 $S$의 $i$번째 문자를 `A`로, $i+1$번째 문자를 `B`로 바꾼다.

준혁이가 주어진 연산을 원하는 만큼 해서 $S$를 $T$로 만들 수 있는지 확인해 보자.

## 입력

첫째 줄에 $N$이 주어진다. $(2 \leq N \leq 300\,000)$

둘째 줄에 `A`와 `B`로만 이루어진 문자열 $T$가 주어진다.

## 출력

첫째 줄에 $S$를 $T$로 만들 수 있다면 `Yes`, 아니라면 `No`를 출력한다.
