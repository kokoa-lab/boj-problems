## 문제

준혁이는 바쁜 하루를 끝내고 와서 Array Lab Of Helpful Alterations(ALOHA)라는 게임을 즐겨 한다.

ALOHA에는 $10 \times 10$의 $2$차원 배열 $A$가 있다. $A$는 $A\_{1,1}$와 $A\_{10,10}$을 배열의 양 끝으로 가진다. 배열의 원소 $A\_{i,j}$는 $-10^9$보다 크거나 같고, $10^9$보다 작거나 같다.

준혁이는 배열을 탐색해 배열에서 최댓값 혹은 최솟값을 찾아야한다. 그러기 위해 준혁이는 자원과 배열 $A$의 값을 담을 배열 $R$이 있다.

준혁이에게는 미네랄과 프로판가스라고 불리는 $2$가지 종류의 자원이 있으며 배열을 탐색하기 위해 배열에 접근할 때 마다 특정 수만큼 미네랄과 프로판가스를 사용한다.

어느날 준혁이는 게임을 하던 도중 모니터가 고장나서 배열 $A$가 보이지 않게 되었다. 당신은 준혁이에게 $A$를 모른 상태에서도 게임을 클리어 할 수 있는 탐색 프로그램을 만들어 달라는 부탁을 받았다.

당신의 프로그램은 다음과 같은 연산을 원하는 만큼 할 수 있다. $t$는 처음에 $1$이다.

* $\max(x,y)$
  + $A\_{x,y}$와 $A\_{x+1,y+1}$를 끝으로 가지는 $2 \times 2$ 사각형 영역에서의 가장 큰 값을 $R\_t$에 넣는다.
  + $1\leq x,y \leq 9$
  + 이 연산 이후 $t$는 $1$증가된다.
  + 미네랄은 $100$, 프로판가스는 $75$ 소모한다.
* $\min(x,y)$
  + $A\_{x,y}$와 $A\_{x+1,y+1}$를 끝으로 가지는 $2 \times 2$ 사각형 영역에서의 가장 작은 값을 $R\_t$에 넣는다.
  + $1\leq x,y \leq 9$
  + 이 연산 이후 $t$는 $1$증가된다.
  + 미네랄은 $100$, 프로판가스는 $75$ 소모한다.
* $\operatorname{plus} (x,y,i)$
  + $A\_{x,y}$를 $R\_i$만큼 증가시킨다.
  + $1\leq x,y \leq 10;1 \leq i < t$
  + 미네랄은 $10$, 프로판가스는 $5$ 소모한다.
* $\operatorname{minus} (x,y,i)$
  + $A\_{x,y}$를 $R\_i$만큼 감소시킨다.
  + $1\leq x,y \leq 10;1 \leq i < t$
  + 미네랄은 $10$, 프로판가스는 $5$ 소모한다.
* $\operatorname{submit} (x,y)$
  + 이 연산이 **수행된 시점**에 $A\_{x,y}$가 $A$의 최솟값 혹은 최댓값과 동일하다는 제출을 한다.
  + 전체 프로그램에서 올바른 $\operatorname{submit}$ 연산이 하나 이상 있어야 정답을 받을 수 있다.
  + $1\leq x,y \leq 10$
  + 미네랄은 $200$, 프로판가스는 $250$ 소모한다.

준혁이는 미네랄을 $4\,000$, 프로판가스를 $3\,000$ 가지고 있다. 사용하는 탐색 프로그램의 모든 연산이 끝날 때 까지 어떤 자원이 $0$미만으로 떨어져서는 안 된다.

배열에서 최대 또는 최소지점을 찾을 수 있는 프로그램을 만들어보자.

## 입력

입력은 없다.

## 출력

첫째 줄에 프로그램이 호출하는 함수의 수 $f$를 출력한다.

그 후 $f$개의 줄에 사용하는 연산들을 한 줄에 하나씩 출력한다.

프로그램의 길이는 제한이 없지만 프로그램의 미네랄과 프로판가스의 사용량이 각각 $4\,000$과 $3\,000$을 넘겨서는 안 된다.
