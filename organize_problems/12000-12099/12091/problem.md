## 문제

이브이는 샤미드(Vaporeon), 쥬피썬더(Jolteon), 부스터(Flareon) 중의 하나로 진화한다.

유저가 제출을 할 때 마다 채점 프로그램은 이브이 세 마리를 진화시킨다. 채점 프로그램은 운이 좋기 때문에, 이브이의 진화 결과는 샤미드, 쥬피썬더, 부스터 한 마리씩 진화한다.

어떤 순서로 진화할지 예측하는 프로그램을 작성하시오. 진화가 실패하는 경우는 없다.

## 입력

첫째 줄에 이브이의 레벨 L (1 ≤ L ≤ 99)이 주어진다. 이브이의 레벨과 진화와는 아무 상관이 없다.

## 출력

첫째 줄에 채점 프로그램이 이브이를 무엇으로 진화시켰는지 출력한다.

* 샤미드: Vaporeon
* 쥬피썬더: Jolteon
* 부스터: Flareon
