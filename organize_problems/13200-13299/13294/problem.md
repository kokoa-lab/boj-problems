## 문제

양의 정수 n이 주어졌을 때 n의 팩토리얼인 n!을 구하는 것은 쉽다. 이번에는 n!이 주어졌을 때 n을 구해 보자.

## 입력

어떤 자연수 n에 대해 n!이 입력으로 주어진다. n!의 자리수는 106 이하이다.

## 출력

n을 출력한다.
