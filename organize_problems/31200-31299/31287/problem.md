## 문제

![달리는 강아지](./001_preview)

`U`, `D`, `L`, `R`로 이루어진 길이 $N$의 문자열 $S$가 주어진다. 문자열 $S$를 $K$번 이어 붙인 문자열을 $T$라고 하자. 장난감 강아지 타카하시는 2차원 좌표평면의 원점에서 시작해서 $T$에 적힌 문자를 하나씩 읽으며, 문자가 `U`라면 위쪽, `D`라면 아래쪽, `L`이라면 왼쪽, `R`이라면 오른쪽으로 한 칸 움직인다.

이때, 타카하시가 $T$를 따라 움직이는 동안 원점에 다시 방문하는 경우가 있는지 구해 보자.

## 입력

첫째 줄에 정수 $N$과 $K$가 공백을 사이에 두고 주어진다. ($1 \le N \le 2\,000,\ 1 \le K \le 10^9$)

둘째 줄에 문자열 $S$가 주어진다. $S$의 모든 문자는 `U`, `D`, `L`, `R`중 하나이다.

## 출력

타카하시가 $T$를 따라 움직이는 동안 원점에 다시 방문하는 경우가 있다면 `YES`를, 없다면 `NO`를 출력한다.
