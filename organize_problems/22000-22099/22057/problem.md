## 문제

Разбирая старые архивы заброшенной библиотеки на краю света, Капитан Женя Водогрей нашел пиратскую карту, указывающую путь к сокровищам. Карта нарисована на куске клетчатой бумаги и хорошо сохранилась. Карта представляет собой невырожденный многоугольник, вырезанный строго по линиям клеток очень большого куска клетчатой бумаги. При этом любая вертикальная и горизонтальная линия пересекает карту не более чем по одному непрерывному отрезку.

Женя готов отправиться в путь за сокровищами, осталось только сложить карту. Чтобы карта не помялась, Женя хочет сложить карту один раз вдоль вертикальной оси, проходящей по границам сетки бумаги, из которой была вырезана карта. Кроме того, чтобы карта не занимала много места, он хочет, чтобы площадь многоугольника, который получится после сложения карты, была минимальна. Помогите ему сделать это. Чтобы Вам помочь, Женя записал координаты каждого угла карты на декартовой плоскости.

![](./001_preview)

На картинке выше слева нарисован пример из условия, а справа нарисован способ достижения ответа — согнуть по оси *x* = 2.

## 입력

Описание карты задается в следующем формате. Первая строка содержит число *n* (1 ≤ *n* ≤ 3·105) — число вершин карты. Следующие *n* строк содержат по два целых числа — координаты этих вершин в порядке обхода по часовой стрелке. Координаты целые, неотрицательные и не превышают 109.

## 출력

В отдельной строке выведите единственное целое число — минимальную площадь после складывания.
