## 문제

![](./001_preview)

![](./002_preview)

gpt가 내준 문제를 풀어보자.

## 입력

첫째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1\leq T \leq 20 000)$

각 테스트 케이스의 첫째 줄에 세 정수 $N$, $X$, $Y$가 공백으로 구분되어 주어진다. $(1\leq N \leq 200 000; 1\leq X,Y \leq 10^9)$

각 테스트 케이스의 둘째 줄에 `R`, `G`, `B`만으로 이뤄진 문자열 $S$가 주어진다.

모든 테스트 케이스에 대해 $N$의 합은 $200 000$을 넘지 않는다.

## 출력

각 테스트 케이스의 첫째 줄에 답을 출력한다.
