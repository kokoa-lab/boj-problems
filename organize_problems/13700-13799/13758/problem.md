## 문제

0보다 크거나 같고, K보다 작거나 같은 수로 이루어진 크기가 N인 배열 중에서 배열의 모든 원소를 XOR한 값이 0보다 큰 배열의 개수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 20,000, 1 ≤ K ≤ 50,000)

## 출력

문제의 조건에 해당하는 배열의 개수를 30011로 나눈 나머지를 출력한다.
