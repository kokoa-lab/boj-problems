## 문제

1부터 N까지 수가 한 번씩 등장하는 순열 P = P1, P2, ..., PN이 있다.

각각의 위치 i가 |Pi - i| = 1을 만족한다면, 그 위치를 좋은 위치라고 한다.

크기가 N인 순열 중에서 좋은 위치가 K개인 순열의 개수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ K ≤ N)

## 출력

크기가 N인 순열 중에서 좋은 위치가 K개인 순열의 개수를 109+7로 나눈 나머지를 출력한다.
