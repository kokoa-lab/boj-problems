## 문제

자연수 n이 주어졌을 때, 1부터 n까지 모든 자연수의 최소공배수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 n (2 ≤ n ≤ 108)이 주어진다.

## 출력

첫째 줄에 1보다 크거나 같고, n보다 작거나 같은 모든 자연수의 최소공배수를 출력한다. 정답이 매우 커질 수 있기 때문에, 232로 나눈 나머지를 출력한다.
