## 문제

방정식 Ax + By + C = 0의 해의 개수를 구하는 프로그램을 작성하시오.

A, B, C, x, y는 모두 정수이고, x1 ≤ x ≤ x2, y1 ≤ y ≤ y2인 해의 개수를 구해야 한다.

## 입력

첫째 줄에 A, B, C, x1, x2, y1, y2가 주어진다. 모든 정수는 -108보다 크거나 같고, 108보다 작거나 같은 정수이다.

## 출력

첫째 줄에 입력으로 주어진 방정식의 해의 개수를 출력한다.
