## 문제

월향 수목원은 최근 $N$개의 식물을 심었다. 초기에 식물들은 아직 영양분을 받지 못한 상태다.

$i$번 식물은 $A\_i$ 만큼의 영양분을 받으면 완전히 성장하며, 완전히 성장한 다음 날부터 구간 $[\max(1,i-R\_i),\min(N,i+R\_i)]$에 속한 식물들에 영양분을 매일 $V\_i$ 만큼 공급하기 시작한다.

월향 수목원의 관리자인 아이보리가 매일 모든 식물에 $1$만큼의 영양분을 준다고 할 때, 모든 식물이 완전히 성장하기 위해서는 적어도 며칠이 걸리는지 구해보자.

## 입력

첫째 줄에 정수 $N$이 주어진다. $(1\leq N \leq 200\,000)$

둘째 줄부터 $N$개의 줄에 걸쳐 정수 $A\_i, R\_i, V\_i$ 가 공백으로 구분되어 주어진다. $(1 \leq A\_i, V\_i \leq 200\,000; 1\leq R\_i \leq N)$

## 출력

모든 식물이 완전히 성장하는 데 필요한 최소 일수를 출력한다.
