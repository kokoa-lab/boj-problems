## 문제

오늘은 하늘이가 공군 기본군사훈련단에 입소하는 날이다.

하늘이가 배정된 대대에는 $a$개의 중대가 있고, 각 중대에는 $b$개의 소대가 있으며, 각 소대에는 정확히 $n$명의 훈련병이 배정된다.

모든 훈련병들은 입소한 순서대로 다음 규칙에 따라 소대에 배정된다.

* $1$중대 $1$소대부터 채운다.
* 한 소대에 $n$명이 모두 채워지면, 같은 중대의 다음 소대로 이동한다.
* 해당 중대의 마지막 소대까지 채워지면, 다음 중대의 첫 번째 소대로 이동한다.

예를 들어 $a=2, b=3$이라면, 소대 배정 순서는 다음과 같다.

1중대 1소대 → 1중대 2소대 → 1중대 3소대 → 2중대 1소대 → 2중대 2소대 → 2중대 3소대

하늘이가 $k$번째로 입소한다고 할 때, 하늘이는 몇 중대 몇 소대에 배정되는지 구하여라.

## 입력

정수 $a,\ b,\ n,\ k$가 공백으로 구분되어 주어진다. ($1 \le a, b, n \le 10$)

하늘이가 소대에 배정되지 못하는 경우는 없다. 즉, $1 \le k \le a \times b \times n$을 만족한다.

## 출력

두 정수 $i,\ j$를 공백으로 구분하여 출력한다.

이는 하늘이가 $i$중대 $j$소대에 배정되었음을 의미한다.
