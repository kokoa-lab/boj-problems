## 문제

1회차(2002년 12월 7일)부터 700회차(2016년 4월 30일)까지 추첨되었던 나눔로또 당첨번호에서 *n*(1 ≤ *n* ≤ 45)이 등장한 횟수를 구하시오. 단, 보너스 번호는 제외한다.

## 입력

*n*(1 ≤ *n* ≤ 45)이 한 줄에 걸쳐 입력으로 주어진다.

## 출력

1회차부터 700회차까지의 당첨번호들(보너스번호 제외) 중 n이 등장한 횟수를 출력한다.
