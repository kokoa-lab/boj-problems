## 문제

Given a set of points with integer coordinates $x\_i, y\_i$, $i = 1 \ldots N$, your program must find all the squares having each of four vertices in one of these points.

## 입력

Input file contains integer $N$ followed by $N$ pairs of integers $x\_i$ $y\_i$.

## 출력

Output file must contain a single integer -- number of squares found.
