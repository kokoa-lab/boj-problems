## 문제

양의 정수 $N$과 $L$에 대해, 다음 조건을 만족하는 수열 $A\_1, A\_2, \cdots, A\_L$을 **제곱근 수열**이라고 한다.

* $A\_1 = N$, $A\_L = 1$
* 모든 $1 \le i < L$에 대해, $\sqrt{A\_i} > A\_{i+1}$이다.
* 모든 항은 양의 정수이다.

$N$과 $L$이 주어질 때, 제곱근 수열의 개수를 구하시오.

## 입력

첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1 \le T \le 100)$

두 번째 줄부터 $T$개의 줄에 걸쳐, 양의 정수 $N$과 $L$이 공백으로 구분되어 주어진다. $(1 \le N \le 70\,000;$ $1 \le L \le 500)$

주어지는 모든 $N$의 합은 $70\,000$ 이하이다.

## 출력

$T$개의 줄에 걸쳐, 각각의 $N$과 $L$에 대해 제곱근 수열의 개수를 출력한다.
