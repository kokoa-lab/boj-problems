## 문제

แวนโก๊ะ ให้บริการวาดภาพให้ลูกค้า โดยผู้ที่มาก่อนจะได้การรับบริการก่อน สมมติให้ที่เริ่มต้น แวนโก๊ะไม่มีงานกับลูกค้าคนใด และเริ่มทํางานที่ t = 0 เมื่อลูกค้าเข้ามาใช้บริการ แวนโก๊ะ จะบันทึกหมายเลขลูกค้า C เวลาที่ลูกค้าคนถัดไปจะเข้ามา I และประเมินเวลาที่จะต้องใช้ในการวาดรูปนั้น S โดยกําหนดให้แวนโก๊ะวาดภาพให้ลูกค้าได้ทีละรูป และจะวาดภาพใหม่ได้ต้องวาดภาพเก่าให้เสร็จก่อนแวนโก๊ะวาดรูปตามลําดับการเข้ามารับบริการของลูกค้า

แวนโก๊ะขอให้คุณเขียนโปรแกรมช่วยแสดงตารางการทํางานเพื่อแสดงเวลาที่งานของลูกค้าเข้ามารอการเริ่มทํางานของแต่ละงาน และเวลาที่แต่ละงานแล้วเสร็จในเวลา 24 ชั่วโมง

## 입력

ตัวเลขแรกคือ n (1<= n<= 20) จํานวนกรณีทดสอบ แต่ละกรณีทดสอบประกอบไปด้วยหลายบรรทัดแต่ละบรรทัดประกอบไปด้วยตัวเลขจํานวนเต็มบวก 3 ตัวเลข คือ หมายเลขลูกค้า C (1 <= C <= 100),เวลาที่ลูกค้าคนถัดไปจะเข้ามา I (1 <= I <= 24) และเวลาที่ให้บริการของลูกค้าคนนั้น S (1 <= S <= 10)กรณีทดสอบสิ้นสุดเมื่อ C=I=S=0

## 출력

สําหรับแต่ละกรณีทดสอบ ให้แสดงตารางการทํางานในหนึ่งวัน (0 - 23) เป็น 24 แถวโดยในแต่ละแถวแสดงตัวเลขจํานวนเต็ม 4 ตัว คือ 1) เวลา 2) หมายเลขลูกค้าที่มาใช้บริการ ณ เวลานั้น 3)หมายเลขลูกค้าของภาพที่แวนโก๊ะ เริ่มวาดในเวลานั้น และ 4. หมายเลขลูกค้าที่งานเสร็จ ณ เวลานั้นหากไม่มีเหตุการณ์เกิดขึ้นให้แสดงผล “-“ แต่ละกรณีทดสอบเว้น 1 บรรทัด
