## 문제

長さ N の文字列 S が与えられる．S の各文字は `o` か `x` のいずれかである．

S に `o` が 3 つ連続している箇所があれば `Yes` を，そうでないならば `No` を， 1 行で出力せよ．

ただし， `o` が 3 つ連続している箇所の隣に `o` があっても良いものとする．

## 입력

入力は以下の形式で与えられる．

```

N
S
```

## 출력

S に `o` が 3 つ連続している箇所があれば `Yes` を，そうでないならば `No` を出力せよ．
