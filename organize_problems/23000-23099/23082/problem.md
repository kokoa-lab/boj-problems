## 문제

균형 삼진법은 밑이 \(3\)이고, 자릿수가 \(0\), \(1\), \(-1\)로 이루어진 기수법이다.

이를 이용해 별도의 부호를 사용하지 않고서도 모든 정수를 유일한 방법으로 나타낼 수 있다.

십진수를 입력 받아 균형 삼진법으로 출력하는 프로그램을 작성하시오.

## 입력

십진법으로 나타낸 정수 \(N\)이 주어진다.

## 출력

문제의 정답을 출력한다. 자릿수가 \(-1\)이라면 대신 '`T`'를 출력한다.

## 힌트

\(10 = 1\times 3^2+0\times 3^1 + 1\times 3^0\)

\(5 = 1\times 3^2+(-1)\times 3^1 + (-1)\times 3^0\)

\(-8 = (-1)\times 3^2+0\times 3^1 + 1\times 3^0\)
