## 문제

양의 정수 $N$이 주어질 때, $\displaystyle\frac{1}{2^N}$을 계산하는 프로그램을 작성하시오.

## 입력

첫째 줄에 정수 $N$이 주어진다.

## 출력

첫째 줄에 $\displaystyle\frac{1}{2^N}$의 정확한 값을 소수 형태로 출력한다. 단, 뒤에 불필요한 $0$을 덧붙여서는 안 된다.
