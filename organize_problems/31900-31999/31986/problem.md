## 문제

수열 $G = \{ G\_1, G\_2, G\_3, \cdots \}$는 Golomb 수열이다. 즉, 다음 조건을 만족하는 유일한 무한수열이다.

* 모든 원소는 양의 정수이다.
* 감소하지 않는 수열이다.
* $G\_i$는 수열 $G$에 존재하는 원소 $i$의 개수이다.

수열 $G$에서 길이가 $N$이면서 마지막 수가 $M$인 증가하는 부분 수열의 개수를 구하는 프로그램을 작성하시오.

## 입력

정수 $N$과 $M$이 공백으로 구분되어 주어진다. $(1 \le N \le M \le 300\,000)$

## 출력

문제의 정답을 $998\,244\,353$으로 나눈 나머지로 출력한다.

## 힌트

$G\_1$, $G\_2$, ..., $G\_9$를 적어보면 다음과 같다.

$1$, $2$, $2$, $3$, $3$, $4$, $4$, $4$, $5$, $\cdots$
