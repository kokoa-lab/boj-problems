## 문제

숫자 $A$, $B$로만 이루어진 양의 정수 $M$개의 합으로 $N$을 나타내어라.

## 입력

총 $T$개의 테스트 케이스가 입력으로 주어지며, 첫 번째 줄에 $T$가 주어진다.

그다음 줄부터 각 테스트 케이스마다 하나의 줄에 양의 정수 $A$, $B$, $M$, $N$이 공백으로 구분되어 주어진다.

## 출력

각 테스트 케이스마다 주어진 순서대로 한 개의 줄에

* 만족하는 식이 존재한다면 $M$개의 수를 공백으로 구분하여 출력한다. 만족하는 식이 여럿인 경우는 그중 아무거나 하나를 출력한다.
* 만족하는 식이 존재하지 않는다면 `-1`을 출력한다.
