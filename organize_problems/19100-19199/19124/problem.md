## 문제

Given $n, k$, calculate $\binom{n}{k} = \frac{n!}{k!(n - k)!} \bmod (2^{32})$.

## 입력

$2$ integers $n, k$ ($1 \leq n \leq 10^{18}, 0 \leq k \leq n$).

## 출력

A single integer denotes the value.
