## 문제

구체적으로 얼마나 많은지 직접 구해보자. 함수 문자열의 정의는 [30047번: 함수 문자열 문제](./001_30047)를 참고하자.

## 입력

첫 번째 줄에 정수 $N$과 $E$가 공백으로 구분되어 주어진다. $(1 \leq N \leq 500;$ $0 \leq E \leq 500)$

## 출력

첫 번째 줄에 길이가 $N$이고 $E$로 평가되는 함수 문자열의 개수를 출력한다. 단, 답이 매우 커질 수 있으므로 $1\,000\,000\,007$로 나눈 나머지를 출력하자.
