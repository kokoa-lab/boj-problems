## 문제

Albert는 용돈 벌이를 위해 주변의 회사 빌딩에서 종종 사내 야식 도시락 배달을 하곤 한다.

빌딩은 총 $F$층이고 총 $N$명의 직원이 야근을 하고 있다. 각 층은 동서로 $W$ 칸, 남북으로 $L$ 칸 격자 모양으로 구성된 개방형 오피스로 총 $W \times L$ 개의 칸으로 나뉘어져있다. 직원들은 1번부터 $N$번까지 번호가 붙어있고, $i$ 번째 직원은 $Z\_i$층에 위치하며 해당 층의 $(X\_i, Y\_i)$ 칸에서 야근을 하고 있다 - 이 때 $1 \le X\_i \le W$ 이고 $1 \le Y\_i \le L$ 이다. Albert는 $S\_Z$층의 $(S\_X, S\_Y)$ 칸에 위치한 사내 편의점에서 $N$개의 도시락을 받은 후, $N$명의 직원에게 도시락을 하나씩 배달해야한다. 층간 이동을 위해서는 각 층의 네 곳 구석에 설치된 에스컬레이터를 이용하면 된다 - 즉, 각 층의 $(1, 1)$, $(W, 1)$, $(1, L)$, $(W, L)$ 칸에 에스컬레이터가 있다.

예를 들어 $F = 5$, $W = 4$, $L = 3$, $N = 4$, $S\_Z = 2$, $S\_X = 1$, $S\_Y = 2$, 그리고 $Z = [1, 1, 5, 5]$, $X = [2, 3, 2, 3]$, $Y = [2, 3, 3, 1]$ 이라 하자. 아래 그림은 각 층의 오피스를 나타내는데, 동서 방향은 좌우로 1번부터 4번칸, 남북 방향은 상하로 1번부터 3번칸을 나타낸다.

* 각 층에서 E로 표시된 네 칸은 윗층이나 아랫층으로 갈 수 있는 에스컬레이터가 설치되어있다.
* $S\_Z = 2$층에 S로 표시된 칸은 편의점이다.
* 1, 2, 3, 4로 표시된 칸은 순서대로 네 명의 직원의 위치이다.

![](./001_preview)

Albert는 편의점에서 도시락을 $N$개 받은 이후 최대한 빨리 모든 직원에게 도시락을 하나씩 배달하고 싶다. Albert가 빌딩 내에서 이동하는데 걸리는 시간은 아래와 같다.

1. 같은 층의 $(x\_1, y\_1)$ 칸에서 $(x\_2, y\_2)$ 칸으로 이동하는데 $|x\_1 - x\_2| + |y\_1 - y\_2|$ 초가 걸린다.
2. 에스컬레이터를 이용하여 한 층 위로 올라가는 경우 2초가 걸린다. 예를 들어 1층의 $(1, 1)$ 에서 2층의 $(1, 1)$ 로 가는데 2초가 걸리고, 마찬가지로 2층의 $(W, L)$에서 3층의 $(W, L)$로 올라가는데도 2초가 걸린다.
3. 에스컬레이터를 이용하여 한 층 아래로 내려가는 경우 1초가 걸린다. 예를 들어 5층의 $(1, L)$에서 4층의 $(1, L)$로 내려가는데 1초가 걸린다.

직원에게 도시락을 전달하기 위해서는 해당 직원이 있는 칸까지 이동해야 하며, 도시락을 전달하는데 걸리는 시간은 0초라하자. 또한, 필요하다면 이동 중에 직원이 일하는 칸을 지나가도 무방하며 마찬가지로 에스컬레이터가 설치된 칸을 지나가도 된다 (즉, 층간 이동을 하지 않고 그냥 지나가도 괜찮다).

위 예제에서 Albert가 4명의 직원에게 도시락을 전달하는 순서는 총 4! = 24가지 가능하다. 그 중 두 가지 방법에 대하여 배달에 소요되는 총 시간을 살펴보면 아래와 같다.

* 방법 1: 1, 2, 3, 4번 순서로 배달하는 경우 총 4 + 2 + 11 + 3 = 20초가 걸린다.
  + 편의점에서 1번 직원에게 가장 빨리 도달하려면 (1, 1) 혹은 (1, 3)에 위치한 에스컬레이터를 이용해야 한다. 2층에서 에스컬레이터까지 1초, 에스컬레이터에서 1초, 1층에서 직원에게 도달하는데 2초를 소요하여 총 4초가 걸린다.
  + 1번 직원에게 배달한 후 2번 직원에게 배달하기 위해 2초가 걸린다.
  + 2번 직원에서 3번 직원에게 배달하려면 (1, 3) 혹은 (4, 3)에 위치한 에스컬레이터를 이용해야한다. 1층과 5층에서 이동하는 시간을 합치면 총 3초가 걸리고, 에스컬레이터로 5층까지 가는데 총 8초가 걸린다 (한 층을 오르는데 2초가 걸린다). 즉, 총 11초가 걸린다.
  + 3번 직원에서 4번 직원에게 이동하는데 3초가 걸린다.
* 방법 2: 2, 4, 1, 3번 순서로 배달하는 경우: 총 4 + 12 + 8 + 11 = 35초가 걸린다.
  + 편의점에서 2번 직원에게 가장 빨리 도달하려면 (1, 3)에 위치한 에스컬레이터를 이용해야 하고 총 4초가 걸린다.
  + 2번 직원에게 배달한 후 4번 직원에게 배달하기 위해 (4, 1) 혹은 (4, 3)에 위치한 에스컬레이터를 이용하면 총 12초가 걸린다.
  + 4번 직원에게 배달한 후 1번 직원에게 배달하기 위해 (1, 1) 혹은 (4, 1)에 위치한 에스컬레이터를 이용하면 되고, 총 8초가 걸린다.
  + 1번 직원에게 배달한 후 3번 직원에게 배달하기 위해 (1, 3) 에 위치한 에스컬레이터를 이용하면 총 11초가 걸린다.

이 예제에서 방법 1이 가장 빠른 시간 안에 모든 직원에게 도시락을 배달할 수 있는 방법이다.

Albert가 모든 직원에게 도시락을 배달하는데 필요한 최소 시간을 구해보자.

## 입력

입력 첫 줄에 테스트 케이스의 수 $T$가 주어진다.

각 테스트 케이스의 첫 줄에는 $F$, $W$, $L$, 그리고 $N$이 공백으로 구분되어 주어진다. 둘째 줄에는 편의점의 위치인 $S\_Z$, $S\_X$, $S\_Y$가 공백으로 구분되어 주어진다. 다음 $N$줄에 걸쳐 각 줄에 3개의 정수가 주어지는데, $i$번째 직원의 위치인 $Z\_i$, $X\_i$, $Y\_i$가 공백으로 구분되어 주어진다.

## 출력

각 테스트 케이스의 정답인 도시락을 모두 배달하는데 드는 최소 시간을 각 줄에 출력한다.
