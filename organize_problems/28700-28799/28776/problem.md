## 문제

У Димы есть устройство, содержащее $n + m$ кнопок и экран. На экране отображается строка $s$, изначально она пустая. Если нажать на кнопку с номером $i$ ($1 \le i \le n$), то в конец строки $s$ допишется строка $w\_i$. Если нажать на кнопку с номером $n + i$ ($1 \le i \le m$), то из конца строки $s$ удалится $a\_i$ символов. Если в строке $s$ было меньше $a\_i$ символов, она станет пустой.

У Димы есть $q$ любимых строк $t\_i$. Для каждой из них он хочет узнать, можно ли в результате нескольких нажатий на кнопки из пустой строки $s$ сделать строку равную $t\_i$.

## 입력

В первой строке дано два целых числа $n$ и $m$ --- количество кнопок первого и второго типа ($1 \le n, m \le 100\,000$).

В следующих $n$ строках даны непустые строки $w\_i$, состоящие из строчных английских букв. Сумма длин всех строк $w\_i$ не превышает $10^6$.

В следующих $m$ строках даны числа $a\_i$ ($1 \le a\_i \le 10^6$).

В следующей строке дано целое число $q$ --- количество любимых строк Димы ($1 \le q \le 100\,000$).

В следующих $q$ строках даны непустые строки $t\_i$, состоящие из строчных английских букв. Сумма длин всех строк $t\_i$ не превышает $10^6$.

## 출력

Для каждой строки $t\_i$ в новой строке выведите <<`Yes`>>, если можно $s$ сделать равным $t\_i$, и <<`No`>> иначе.
