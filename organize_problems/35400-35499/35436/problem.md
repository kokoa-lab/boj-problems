## 문제

어느 날 뉴로는 길이 $N$의 수열 $A, B$를 발견했다. 각 수열의 원소는 $10^5$ 이하의 음이 아닌 정수로 이루어져 있었고, 수열은 아름다운 색을 띠고 있었다. 뉴로는 두 수열 $A,B$의 **다채로움**을 판단하기 위해, 두 수열의 **다채로움**을 다음과 같이 정의했다.

$\max\limits\_{1 \le i, j \le N} (A\_i\cdot B\_j - B\_i\cdot A\_j)$

하지만 사악한 이블이 수열을 마음대로 바꿔버렸다! 이블은 총 $Q$번의 **변형**을 수행했으며, $i$번째 변형에서는 처음에 주어졌던 두 수열의 모든 원소에 어떤 정수 $k\_i$를 더했다. 뉴로를 도와 각 변형이 적용된 수열의 **다채로움**을 계산하자.

각 변형은 이전 변형의 결과에 영향을 받지 않는다. 즉, 모든 변형은 서로 독립적이다.

## 입력

첫 번째 줄에 수열의 길이 $N$이 주어진다.

두 번째 줄에 수열 $A$의 원소 $A\_1, A\_2, \cdots, A\_N$이 공백으로 구분되어 주어진다.

세 번째 줄에 수열 $B$의 원소 $B\_1, B\_2, \cdots, B\_N$이 공백으로 구분되어 주어진다.

네 번째 줄에 이블이 수열을 변형한 횟수 $Q$가 주어진다.

다섯 번째 줄부터 $Q$개의 줄에 걸쳐 $k\_i$가 주어진다.

## 출력

총 $Q$개의 줄에 걸쳐, $i$번째 줄에 $i$번째 변형을 적용했을 때의 **다채로움**을 출력하라.
