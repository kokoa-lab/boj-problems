## 문제

길이가 $M$인 수열 $S$의 점수는 다음과 같이 정의된다.

$$score(S) = \sum\_{1 \le i < j \le M} S\_i \cdot S\_j$$

길이가 $N$인 수열 $A\_1, A\_2, \cdots, A\_N$이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.

* $1\ i\ v$ : $A\_i = v$를 수행한다.
* $2\ l\ r\ k$ : 구간 $[l,r]$에 속한 원소들을 $k$가지 색깔 중 하나로 칠한다. 각 색깔은 최소 한 번 이상 사용되어야 한다. 색깔이 $c$인 모든 원소들로 이루어진 부분 수열을 $B\_c$라고 할 때, 가능한 모든 색칠 방법 중 다음 수식의 최댓값을 출력한다. $$\sum\_{c = 1}^k score(B\_c)$$

## 입력

첫 번째 줄에 수열의 크기 $N$이 주어진다.

두 번째 줄에 $A\_1, A\_2, \cdots, A\_N$이 공백으로 구분되어 주어진다.

세 번째 줄에 쿼리의 개수 $Q$가 주어진다.

네 번째 줄부터 $Q$개의 줄에 걸쳐 쿼리가 한 줄에 하나씩 주어진다.

## 출력

$2$번 쿼리가 주어질 때마다 정답을 줄 바꿈으로 구분하여 출력하라.
