## 문제

다음 조건을 만족하는 두 양의 정수 $(A, B)$를 좋은 쌍으로 정의합니다.

* $A < B$ 입니다.
* $B - A = N$ 입니다.
* $A \times B$ 의 약수의 개수는 정확히 $2$개입니다.

정수 $N$이 주어졌을 때, 좋은 쌍 $A, B$를 모두 찾으십시오.

## 입력

첫째 줄에 테스트 케이스의 개수 $T$가 주어집니다. ($1 \leq T \leq 1\,000$)

둘째 줄부터 $T$개의 줄에 걸쳐 정수 $N$이 주어집니다. ($1 \leq N \leq 10\,000$)

## 출력

각 테스트 케이스의 첫째 줄에 좋은 쌍의 개수 $K$를 출력합니다.

다음 $K$개의 줄에 걸쳐, 조건을 만족하는 좋은 쌍 $(A, B)$를 $A$에 대한 오름차순으로 한 줄에 한 쌍씩 공백으로 구분하여 출력합니다.

모든 테스트 케이스에 대한 좋은 쌍의 개수의 합은 $10^6$을 넘지 않음이 보장됩니다.
