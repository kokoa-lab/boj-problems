## 문제

다음 조건을 만족하는 길이 $N$의 수열 $P\_{1}, P\_{2}, \cdots, P\_{N}$을 구해보자.

* 각 원소 $P\_i$는 $150$ 이하의 소수이다. $(1\leq i \leq N)$
* 수열의 각 원소의 제곱을 모두 더한 값, 즉 ${P\_{1}}^{2} + {P\_{2}}^{2} + \cdots + {P\_{N}}^{2}$이 수열에 포함된 모든 원소 $P\_i$로 나누어떨어져야 한다.
* 수열 안의 원소가 모두 같아서는 안 된다. 다시 말해 ${P\_{1}} = {P\_{2}} = \cdots = {P\_{N}}$이어서는 안 된다.

## 입력

첫 번째 줄에 정수 $N$이 주어진다.

## 출력

만약 조건을 만족하는 수열이 있다면 첫 번째 줄에 수열 $P\_1, P\_2, \cdots, P\_{N}$을 공백으로 구분하여 출력한다. 가능한 수열이 여러 개라면 그중 아무것이나 출력한다.

만약 조건을 만족하는 수열이 없다면 첫 번째 줄에 `-1`을 출력한다.
