## 문제

N개의 노드로 이루어진 회로가 있다. 이 회로에 저항은 총 M개가 있다.

회로의 정보가 주어졌을 때, 1번 노드와 N번 노드 사이의 총 저항값을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 노드의 개수 N (2 < N ≤ 100)과 저항의 개수 (1 ≤ M ≤ 300)가 주어진다.

둘째 줄부터 M개의 줄에는 저항의 정보가 주어진다. 저항의 정보는 세 정수 u, v, r로 나타낼 수 있다. u와 v는 저항이 연결하는 두 노드의 번호이고, r은 저항값이다. (1 ≤ u, v ≤ N, u ≠ v, 1 ≤ r ≤ 100)

## 출력

1번과 N번 노드 사이의 총 저항을 출력한다. 1번과 N번 사이의 총 저항이 무한대인 경우는 없다. 절대/상대 오차는 10-6까지 허용한다.
