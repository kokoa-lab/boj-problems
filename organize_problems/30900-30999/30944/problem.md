## 문제

길이가 N인 수열 A1, A2, ..., AN이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.

* m k: A1, A2, ..., Am 의 부분수열 (subsequence) 중 [최장 증가 부분수열](./001_12738)의 길이가 k 이하인 것의 최대 길이를 출력하라. (1 ≤ k ≤ m ≤ N)

## 입력

첫째 줄에 수열의 크기 N과 쿼리의 개수 Q가 주어진다. (1 ≤ N ≤ 50,000, 1 ≤ Q ≤ 200,000)

둘째 줄에는 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 50,000)

이후 Q개의 줄에 위에서 설명한 것과 같은 쿼리가 주어진다.

## 출력

각 쿼리에 대해 정답을 한 줄에 출력하라.
