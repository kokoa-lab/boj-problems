## 문제

a, b, c가 주어졌을 때, A \* B = C 이면서 |A - a| + |B - b| + |C - c|의 최솟값을 구하는 프로그램을 작성하시오. 이때, A, B, C는 양의 정수이어야 한다.

|x|는 x의 절댓값이다.

## 입력

첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 1,000,000,000)가 주어진다.

## 출력

첫째 줄에 |A - a| + |B - b| + |C - c|의 최솟값을 출력한다. 이때, A\*B는 C와 같아야 한다.
