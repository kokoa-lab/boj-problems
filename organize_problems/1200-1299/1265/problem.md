## 문제

1부터 N까지의 수를 임의로 배열한 순열은 총 N! = N×(N-1)×…×2×1 가지가 있다.

모든 1 ≤ i ≤ N에 대해서 |P[i] - i| ≤ K를 만족하는 순열 P의 개수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N, K가 순서대로 주어진다. N은 100보다 작거나 같은 자연수이고, K는 6보다 작거나 같은 자연수이다.

## 출력

첫째 줄에 문제의 정답을 출력한다.
